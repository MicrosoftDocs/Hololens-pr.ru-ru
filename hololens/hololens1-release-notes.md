# <a name="hololens-1st-gen-release-notes"></a>заметки о выпуске HoloLens 1 (gen)

## <a name="hololens-1st-gen-long-term-servicing"></a>долгосрочное обслуживание HoloLens (1-й общий)
HoloLens (первое поколение) перешла в состояние долгосрочного обслуживания (LTS). в будущих обновлениях основное внимание уделяется устранению проблем и исправлений безопасности, а Windows 10 Holographic также обеспечению соответствия функций версии 1809 для HoloLens (1-го поколения).

для разработчиков это означает, что HoloLens (1 gen) приложения не будут поддерживать API опенкср.  Эти гарнитуры по-прежнему поддерживаются в Unity 2019 LTS с серверной частью API WinRT для полного жизненного цикла Unity 2019 LTS до середины 2022.

### <a name="windows-10-version-1803-for-microsoft-hololens"></a>Windows 10, версия 1803 для Microsoft HoloLens

> область **применения:** HoloLens (1-й общий)

Windows 10, версия 1803, является первым обновлением компонента для Windows Holographic for Business с момента выпуска в Windows 10 версии 1607. В этом обновлении представлены следующие изменения.

- ранее вы могли только убедиться, что лицензия на обновление для коммерческого набора была применена к устройству HoloLens. для этого проверьте, был ли VPN доступен на устройстве. теперь **Параметры**  >  **система** будет отображать **Windows Holographic for Business** после применения лицензии на обновление. [узнайте, как разблокировать функции Windows Holographic for Business](hololens1-upgrade-enterprise.md).

- номер сборки операционной системы можно просмотреть в свойствах устройства в приложении проводника и в [средстве Windows device Recovery Tool (вдрт)](https://support.microsoft.com/help/12379/windows-10-mobile-device-recovery-tool-faq).
- подготовка HoloLens устройства теперь упрощается с помощью мастера создания **HoloLens устройств** в средстве конструктора конфигураций Windows. В мастере можно настроить интерфейс установки и сетевые подключения, задать режим разработчика и получить негрупповые маркеры Azure AD. [Узнайте, как использовать мастер простой подготовки для HoloLens](hololens-provisioning.md#provisioning-package-hololens-wizard).

- При создании локальной учетной записи в пакете подготовки пароль больше не будет истечь каждые 42 дней.

- HoloLens можно [настроить как киоск с одним приложением или с несколькими приложениями](hololens-kiosk.md). режим киоска с несколькими приложениями позволяет настроить HoloLens для запуска только указанных вами приложений и запрещает пользователям вносить изменения.

- протокол передачи мультимедиа (MTP) включен, чтобы можно было подключить устройство HoloLens к компьютеру по USB и передавать файлы между HoloLens и компьютером. Можно также использовать приложение проводника для перемещения и удаления файлов в HoloLens.

- ранее после входа на устройство с помощью учетной записи Azure Active Directory (Azure AD) вам пришлось **добавить рабочий доступ** в **Параметры** , чтобы получить доступ к корпоративным ресурсам. Теперь вы входите с учетной записью Azure AD, и регистрация выполняется автоматически.

- Перед входом в систему можно выбрать значок сети под полем "пароль", чтобы выбрать другую Wi-Fi сеть для подключения. Вы также можете подключиться к гостевой сети, например в отеле, в центре конференц-связи или в компании.

- теперь вы можете легко [предоставлять общий доступ к HoloLens нескольким людям](hololens-multiple-users.md) с помощью учетных записей Azure AD.

- При сбое установки или входа выберите новый параметр **сбор сведений** , чтобы получить журналы диагностики для устранения неполадок.

- Отдельные пользователи могут синхронизировать корпоративную электронную почту без регистрации устройства в системе управления мобильными устройствами (MDM). Вы можете использовать устройство с учетной записью Майкрософт, скачать и установить почтовое приложение и добавить учетную запись электронной почты напрямую.

- вы можете проверить состояние синхронизации MDM для устройства в **Параметры**  >  **учетных записей**  >  **доступ к рабочей или учебной**  >  **информации**. В разделе **состояние синхронизации устройства** можно запустить синхронизацию, Просмотреть области, управляемые MDM, а также создать и экспортировать Расширенный отчет о диагностике.
