# Заметки о выпуске для HoloLens 1 (Gen)

## Долгосрочное обслуживание HoloLens (1-го поколения)
HoloLens (1-го поколения) ввел состояние долгосрочного обслуживания (LTS). Будущие обновления будут сосредоточены на проблемах и исправлениях безопасности, сохраняя четность функций с помощью выпуска Windows 10 holographic, версия 1809 для HoloLens (1-го поколения).

Для разработчиков это означает, что приложения HoloLens (1-го поколения) не будут поддерживать API OpenXR.  Эти гарнитуры поддерживаются в Unity 2019 LTS с серверным внутренним интерфейсом WinRT API для полного жизненного цикла в Unity 2019 LTS с помощью MID-2022.

### Windows 10 версии 1803 для Microsoft HoloLens

> **Применимо к:** HoloLens (1-го поколения)

Windows 10 версии 1803 — первое обновление компонентов для Windows holographic для бизнеса с момента его выпуска в Windows 10 версии 1607. Это обновление вносит следующие изменения:

- Ранее вы могли только убедиться, что лицензия на обновление для коммерческого набора была применена к устройству HoloLens, проверяя, был ли VPN доступен на устройстве. **Settings**  >  После применения лицензии на обновление**система** параметров отобразит **Windows holographic для бизнеса** . [Сведения о том, как разблокировать Windows holographic для бизнес-функций](hololens1-upgrade-enterprise.md).

- Номер сборки операционной системы можно просмотреть в свойствах устройства в приложении "Проводник" и в [средстве восстановления Windows Device (WDRT)](https://support.microsoft.com/help/12379/windows-10-mobile-device-recovery-tool-faq).
- Подготовить устройство HoloLens стало проще с помощью нового мастера **подготовки устройств hololens** в конструкторе конфигураций Windows. В мастере вы можете настроить параметры настройки и подключения к сети, задать режим разработчика и получить массовые маркеры Azure AD. [Сведения о том, как использовать простой мастер подготовки для HoloLens](hololens-provisioning.md#provisioning-package-hololens-wizard).

- Когда вы создаете локальную учетную запись в пакете подготовки, срок действия пароля больше не истечет через 42 дней.

- HoloLens можно [настроить как одно приложение или многоприложениеное киоск](hololens-kiosk.md). Режим киоска с несколькими приложениями позволяет настроить HoloLens на запуск только указанных вами приложений и запрещает пользователям вносить изменения.

- Протокол передачи мультимедиа (MTP) включен, и вы можете подключить устройство HoloLens к компьютеру с помощью USB и передать файлы между HoloLens и компьютером. Вы также можете использовать приложение "Проводник" для перемещения и удаления файлов в HoloLens.

- Ранее после входа в систему на устройстве с помощью учетной записи Azure Active Directory (Azure AD), вам пришлось **Добавить доступ** к ресурсам Организации в **Параметры** , чтобы получить доступ к ним. Теперь вы входите в систему с помощью учетной записи Azure AD, и регистрация происходит автоматически.

- Перед входом вы можете выбрать значок сети под полем пароля, чтобы выбрать другую Wi-Fi сеть для подключения. Вы также можете подключиться к гостевой сети, например в отеле, в центре конференции или в компании.

- С помощью учетных записей Azure AD вы можете легко [предоставить общий доступ к HoloLens нескольким людям](hololens-multiple-users.md) .

- В случае сбоя при установке или входе выберите новый параметр **сбора сведений** , чтобы получить журналы диагностики для устранения неполадок.

- Пользователи могут синхронизировать корпоративную электронную почту без регистрации их устройства в службе управления мобильными устройствами (MDM). Вы можете использовать устройство с учетной записью Майкрософт, скачать и установить приложение "почта" и добавить учетную запись электронной почты напрямую.

- Вы можете проверить состояние синхронизации MDM для устройства в разделе **настройки**  >  **учетных записей**  >  .**доступ к**  >  **сведениям о**работе или учебе. В разделе " **состояние синхронизации устройства** " можно начать синхронизацию, Просмотреть области, управляемые MDM, а также создать и экспортировать отчет с расширенной диагностикой.
