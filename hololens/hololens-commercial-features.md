# <a name="hololens-2-capabilities-and-solutions"></a>Возможности и решения HoloLens 2

## <a name="why-hololens"></a>В чем преимущества HoloLens?

С помощью HoloLens 2 и приложений смешанной реальности сотрудники теперь могут продуктивно работать совместно из любой точки и точно выполнять задания.

Работайте совместно без ограничений и действуйте точно, чтобы повысить продуктивность сотрудников, с приложениями смешанной реальности на HoloLens 2. Концентрируйтесь на решаемой задаче дольше и эффективнее и освободите руки благодаря встроенным голосовым командам, отслеживанию глаз и привязке голограмм к объектам реального мира, позволяющим безопасно выполнять задачи без ошибок. Связывайтесь с удаленными коллегами в режиме реального времени и работайте совместно на обширном голографическом холсте, накладываемом на физическую среду, чтобы быстро решать рабочие задачи. Оцените реализуемую без предварительной подготовки рентабельность инвестиций с надежной экосистемой приложений, для которых корпорация Майкрософт обеспечивает безопасность, надежность и масштабируемость.  

Надеемся, что эта статья поможет вам оценить различные преимущества смешанной реальности для бизнеса.

## <a name="hololens-2-capabilities"></a>Возможности HoloLens 2

Что делает устройство HoloLens 2 таким мощным?

| Функция | Описание |
|---------|-------------|
| Привязка голограмм к объектам реального мира | Привязанные голограммы остаются точно на месте. HoloLens 2 распознает ваше рабочее пространство. Поэтому цифровое содержимое остается на месте, привязанное к объектам или поверхностям, с которыми вы работаете. |
| Отслеживание рук | Вы можете трогать, захватывать и перемещать голограммы естественным образом. HoloLens 2 адаптируется к движениям ваших рук, предоставляя новый уровень удобства в работе. |
| Отслеживание взгляда | Насладитесь новым уровнем понимания контекста и человека. HoloLens 2 точно определяет направление взгляда и может понять ваши намерения и адаптировать голограммы к положению глаз в режиме реального времени. |
| Поддержка голосовых команд | Встроенные голосовые команды позволяют быстро выполнять навигацию и работать с HoloLens 2, когда ваши руки заняты. |
| Эргономичность | Устройство HoloLens 2 имеет небольшой вес (3,28 кг) и включает систему подгонки размера для длительного и комфортного ношения. |
| Широкое поле зрения | Расширьте свой голографический холст благодаря дисплеям высокого разрешения с широким полем зрения. |
| Никаких препятствий | Перемещайтесь свободно, без проводов и внешних модулей, мешающих в работе. |
| На базе Azure | Выполняйте потоковую передачу трехмерного содержимого высокой точности, которое может быть привязано к расположению и/или объекту, доступному для пользователей в службах смешанной реальности Azure.
| Смешанный захват реальности | Документируйте свой опыт в фотографиях и видео и делитесь им с другими в реальном времени. |
| Windows Hello для бизнеса | Биометрическая проверка подлинности на основе радужной оболочки глаза позволяет быстро и безопасно влиться в работу. |
| Windows Autopilot | Выберите и предварительно настройте службы для HoloLens 2, чтобы они сразу же были готовы к использованию на распределенных рабочих местах. |
| обновления ОС; | Поддерживайте защиту с помощью ежемесячных обновлений для обслуживания и воспользуйтесь новыми возможностями для обеспечения производительности и управляемости в выпусках, выходящих дважды в год. |
| Простое управление устройствами | Управляйте несколькими устройствами HoloLens 2 одновременно с помощью таких решений как Microsoft Intune, VMware Workspace One, MobileIron и многих других. |
| Работа в регулируемых средах | HoloLens 2 включает обширное портфолио устройств, поддерживающее среды с высокой степенью регулирования, в том числе среды, соответствующие ISO класса 5.0 и UL класса I, раздел 2. |

## <a name="managing-hololens-2-in-your-organization"></a>Управление HoloLens 2 в организации

HoloLens 2 включает в себя функции, которые упрощают управление устройствами HoloLens и их использование в организациях. Некоторые функции входят в состав устройства, а другие можно включить с помощью [Управления мобильными устройствами (MDM) для HoloLens](hololens-mdm-configure.md) или путем [подготовки пакетов](hololens-provisioning.md) с использованием [Конструктора конфигураций Windows](app-deploy-provisioning-package.md#setup).

| Я хочу... | Решение | Описание |  
|---------| ------------|------------|
Управление входом конечных пользователей | [**Идентификация**](hololens-identity.md) | HoloLens 2 поддерживает несколько типов удостоверений пользователей: Azure Active Directory (AAD), учетная запись Майкрософт (MSA) и локальные учетные записи.  |
| Шифрование данных пользователей | [**Безопасность данных**](security-encryption-data-protection.md) | На HoloLens 2 включено шифрование данных BitLocker, которое обеспечивает такой же уровень защиты, как на любом устройстве под управлением Windows. | 
Управление устройствами HoloLens в моей организации | [**Управление мобильными устройствами (MDM)**](hololens-mdm-configure.md) | Управляйте параметрами, выбирайте приложения для установки и настраивайте конфигурации безопасности, адаптированные к потребностям организации, на нескольких устройствах HoloLens 2 из центрального расположения. | 
|Минимизация времени установки для новых пользователей и устройств | [**Autopilot**](hololens2-autopilot.md) | Настройте интерфейс запуска при первом включении (OOBE) в Microsoft Endpoint Manager и разрешите пользователям подготавливать устройства для коммерческого использования с минимальным взаимодействием или без него. |  
| Управление обновлениями ОС на устройствах | [**Центр обновления Windows для бизнеса**](hololens-updates.md#managing-updates-by-using-windows-update-for-business) | Центр обновления Windows для бизнеса предоставляет контролируемые обновления операционной системы для устройств и поддержку канала долгосрочного обслуживания. |  
| Разрешение скачивания конкретных приложений и бизнес-приложений |[**Управление приложениями**](app-deploy-overview.md) | Выберите способ распространения и контроля приложений для выбранных групп пользователей HoloLens 2. | 
| Отображение или скрытие конкретных приложений в меню "Пуск" |[**Полноэкранный режим**](hololens-kiosk.md) | Настройте HoloLens 2 для работы в качестве устройства с фиксированным назначением для использования в демонстрационных приложениях или в специализированных бизнес-приложениях. 
| Защита среды путем блокировки приложений | [**WDAC**](windows-defender-application-control-wdac.md) | Управление приложениями в Защитнике Windows (WDAC) блокирует запуск приложений и процессов пользователем устройства.
| Управление безопасностью устройств с помощью правил для приложений и процессов | [**Политики (участники программы CSP)**](hololens-csp-policy-overview.md) | ИТ-администраторы могут определять и реализовывать параметры политики, используя существующий список поддерживаемых поставщиков служб настройки политики на HoloLens 2. |  
| Управление тем, как устройство подключается к Интернету | [**Сеть и подключения**](hololens-certificates-network.md) | Используйте проверку подлинности на основе сертификатов для доступа к Wi-Fi, VPN или внутренним ресурсам. | 
| Предоставление общего доступа к устройству нескольким пользователям | [**Автоматическая настройка дисплея**](hololens-calibration.md#auto-eye-position-support) | Дисплей HoloLens 2 настраивается автоматически с помощью алгоритма автоматического определения положения глаз (AEP), что исключает необходимость в ручной калибровке при [передаче устройства от одного пользователя другому](hololens-multiple-users.md). |

Ознакомьтесь с [требованиями к лицензированию](hololens-licenses-requirements.md) для перечисленных выше решений.
