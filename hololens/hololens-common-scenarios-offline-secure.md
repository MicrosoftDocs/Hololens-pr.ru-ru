# <a name="common-scenarios--offline-secure-hololens-2"></a>Распространенные сценарии — автономные безопасные устройства HoloLens 2

## <a name="overview"></a>Обзор

В этом руководстве приводится руководство по использованию примера пакета предварительного обеспечения, который блокирует HoloLens 2 для использования в безопасных средах со следующими ограничениями:

-   Отключение Wi-Fi.
-   Отключить BlueTooth.
-   Отключить микрофоны.
-   Предотвращает добавление или удаление пакетов подготовка.
-   Ни один пользователь не может включить любой из вышеуказанных компонентов с ограниченным доступом.

## <a name="prepare"></a>Подготовка

Настройка ПК в Windows 10
1. [Скачайте последний os-файл HoloLens 2](https://aka.ms/hololens2download) непосредственно на компьютер. 
   1. Поддержка этой конфигурации включена в сборку 19041.1117 и выше.
1. Загрузка и установка средства Advanced Recovery Companion (ARC) из [Microsoft Store](https://www.microsoft.com/store/productId/9P74Z35SFRS8) на компьютер
1. Скачивание и установка последнего средства конструктора конфигурации [Windows (WCD)](https://www.microsoft.com/p/windows-configuration-designer/9nblggh4tx22?activetab=pivot:overviewtab) из Microsoft Store на компьютер.
1. [Загрузите OfflineSecureHL2_Sample папку с файлами проекта для](https://aka.ms/HoloLensDocs-SecureOfflineSample) создания PPKG.
1. Подготовка автономного [приложения Line of Business для развертывания PPKG.](app-deploy-provisioning-package.md) 


## <a name="configure"></a>Настройка

Создание пакета обеспечения безопасной конфигурации

1. Запустите средство WCD на компьютере.
1. Выберите **файл -> Open project**.
  1. Перейдите к расположению сохраненной ранее OfflineSecureHL2_Sample и выберите: OfflineSecureHL2_Sample.icdproj.xml
1. Проект должен открыться, и теперь у вас должен быть список доступных настроек:

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана пакета конфигурации, открытого в WCD](images/offline-secure-sample-wcd.png)

   Конфигурации, заданной в этом пакете подготовка:
   
   |     Элемент                                                |     Параметр                       |     Описание                                                                                                                    |
   |---------------------------------------------------------|-----------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
   |     Учетные записи / Пользователи                                    |     Имя локального пользователя & пароль    |     Для этих автономных устройств необходимо установить одно имя пользователя и пароль и делиться им со всеми пользователями устройства.          |
   |     Первый опыт / HoloLens / SkipCalibration       |     True                          |     Пропускает калибровку только при начальной настройке устройства                                                                             |
   |     First Experience / HoloLens / SkipTraining          |     True                          |     Пропускает обучение устройств во время начальной установки устройства                                                                              |
   |     First Experience / HoloLens / WiFi                  |     True                          |     Пропускает Wi-Fi при начальной настройке устройства                                                                                 |
   |     Политики/Подключение/AllowBluetooth                |     Нет                            |     Отключение Bluetooth                                                                                                             |
   |     Политики/Опыт/AllowCortana                    |     Нет                            |     Отключение Кортаны (для устранения потенциальных проблем с отключением микрофонов)                                          |
   |     Политики/Смешанная реальность/MicrophoneDisabled            |     Да                           |     Отключение микрофона                                                                                                            |
   |     Политики/Конфиденциальность/LetAppsAccessLocation              |     Отказ в силе                    |     Предотвращает попытки приложений получить доступ к данным о местоположении (чтобы устранить возможные проблемы с отключением отслеживания местоположения)    |
   |     Политики/Конфиденциальность/LetAppsAccessMicrophone            |     Отказ в силе                    |     Предотвращает попытки приложений получить доступ к микрофонам (чтобы устранить возможные проблемы с отключением микрофонов)           |
   |     Политики/Безопасность/AllowAddProvisioningPackage       |     Нет                            |     Предотвращает добавление пакетов подготовка, которые могут пытаться переопределять заблокированные политики.                         |
   |     Политики/безопасность/AllowRemoveProvisioningPackage    |     Нет                            |     Предотвращает удаление этого заблокированного пакета предварительного обеспечения.                                                           |
   |     Политики/Система/AllowLocation                       |     Нет                            |     Предотвращает попытки устройства отслеживать данные о расположении.                                                                        |
   |     Политики/WiFi/AllowWiFi                             |     Нет                            |     Отключение Wi-Fi                                                                                                                 |

1. В соответствии с настройками времени запуска выберите **учетные записи / Пользователи / Имя пользователя: Holo / Пароль**.

   Обратите внимание на пароль и сброс при желании.

1. Перейдите в UniversalAppInstall / UserContextApp и настройте приложение [LOB,](app-deploy-provisioning-package.md) которое будет развернуто на этих устройствах.

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана, где добавить приложение в WCD.](images/offline-secure-sample-wcd-usercontextapp2.png)

1. После завершения выберите кнопку "Экспорт" и выполните все запросы, пока не будет создан пакет подготовка.

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана кнопки Экспорт для этого пакета в WCD.](images/offline-secure-sample-wcd-export.png)

## <a name="deploy"></a>Развертывание

1. Подключите HL2 к компьютеру Windows 10 с помощью USB-кабеля.
1. Запустите инструмент ARC и выберите **HoloLens 2**

   ![Первоначальный экран установки чистого образа HoloLens2](images/ARC2.png)

1. На следующем экране выберите **ручной выбор пакета.**

   ![Информационный экран HoloLens 2 ARC](images/arc_device_info.png)

1. Перейдите к ранее загруженным файлу .ffu и выберите **Open**.
1. На странице Предупреждение выберите **Продолжить**.

   ![Экран предупреждения HoloLens 2 ARC](images/arc_warning.png)

1. Подождите, пока инструмент ARC завершит установку ОС HoloLens 2.
1. После того как устройство завершит установку и сапоги обратно, с компьютера перейдите на файл Explorer и скопируйте ранее сохраненный файл PPKG в папку устройства.

   > [!div class="mx-imgBorder"]
   > ![Файл PPKG на компьютере в окне File Explorer.](images/offline-secure-file-explorer.png)

1. На holoLens 2 нажмите следующую комбо кнопки, чтобы **** запустить пакет подготовка: нажмите кнопку Вниз громкости и **power Button** одновременно.
1. Вам будет предложено применить пакет подготовка, выберите **Подтверждение**
1. После завершения пакета подготовка выберите **ОК**.
1. После этого вам будет предложено войти в устройство с общей локальной учетной записью и паролем.

## <a name="maintain"></a>Обслуживание

В этой конфигурации рекомендуется перезапустить процесс выше и перезапустить устройство с помощью инструмента ARC и применить новый PPKG для обновления оси и/или приложения(ы).
