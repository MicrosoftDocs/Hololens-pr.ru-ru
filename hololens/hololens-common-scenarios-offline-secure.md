# Распространенные сценарии: автономная служба Secure HoloLens 2

## Обзор
Это руководство содержит рекомендации по применению примера пакета подготовки, который блокирует использование HoloLens 2 для использования в защищенных средах со следующими ограничениями:
-   Отключите WiFi.
-   Отключите BlueTooth.
-   Отключить микрофоны.
-   Запрещает добавление и удаление пакетов подготовки.
-   Ни один пользователь не может включить ни один из перечисленных выше компонентов.

## Подготовка 
Настройка компьютеров с Windows 10
1. [Загрузи последнюю версию файла операционной системы HoloLens 2](https://aka.ms/hololens2download) на компьютер напрямую. 
   1. Поддержка этой конфигурации включена в сборку 19041,1117 и более поздние версии.
1. Скачивание и установка средства "расширенного помощника по восстановлению" [из магазина Microsoft Store](https://www.microsoft.com/store/productId/9P74Z35SFRS8) на компьютер
1. Скачайте и установите последнюю версию средства [разработки конфигурации Windows (WCD)](https://www.microsoft.com/p/windows-configuration-designer/9nblggh4tx22?activetab=pivot:overviewtab) из магазина Microsoft Store на компьютер.
1. [Скачайте папку OfflineSecureHL2_Sample с файлами проекта](https://aka.ms/HoloLensDocs-SecureOfflineSample) , чтобы создать PPKG.
1. Подготовка автономной [строки бизнес-приложения для PPKG развертывания](app-deploy-provisioning-package.md). 


## Настройка
Создание пакета обеспечения безопасной конфигурации

1. Запустите средство WCD на компьютере.
1. Выберите **файл-> открыть проект**.
  1. Перейдите к расположению ранее сохраненной OfflineSecureHL2_Sample папки, а затем выберите: OfflineSecureHL2_Sample.icdproj.xml
1. Проект должен открыться, и теперь у вас должен быть список доступных настроек. 

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана: пакет конфигурации открыт в WCD](images/offline-secure-sample-wcd.png)

Конфигурации, настроенные в этом пакете подготовки:

|     Элемент                                                |     Параметр                       |     Описание                                                                                                                    |
|---------------------------------------------------------|-----------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
|     Учетные записи и пользователи                                    |     Имя локального пользователя & паролем    |     Для этих автономных устройств необходимо настроить одно имя пользователя и пароль, чтобы они были доступны всем пользователям устройства.          |
|     Первый опыт/HoloLens/SkipCalibration       |     True                          |     Пропуск калибровки только на начальном этапе настройки устройства                                                                             |
|     Первый опыт/HoloLens/SkipTraining          |     True                          |     Пропускает обучение устройства во время первоначальной настройки устройства                                                                              |
|     Первый опыт/HoloLens/WiFi                  |     True                          |     Пропускает конфигурацию Wi-Fi во время первоначальной настройки устройства.                                                                                 |
|     Политики/подключение/AllowBluetooth                |     Нет                            |     Отключение Bluetooth                                                                                                             |
|     Политики/опыт/AllowCortana                    |     Нет                            |     Отключение Кортаны (для устранения потенциальных проблем, так как микрофон отключен)                                          |
|     Политики/MixedReality/MicrophoneDisabled            |     Да                           |     Отключение микрофона                                                                                                            |
|     Политики/конфиденциальность/LetAppsAccessLocation              |     Запретить принудительно                    |     Приложение не пытается получить доступ к данным о расположении (для устранения потенциальных проблем, так как отслеживание расположения отключено)    |
|     Политики/конфиденциальность/LetAppsAccessMicrophone            |     Запретить принудительно                    |     Запрещает приложениям получать доступ к микрофонам (для устранения потенциальных проблем, так как микрофон отключен)           |
|     Политики/безопасность/AllowAddProvisioningPackage       |     Нет                            |     Запрещает любому пользователю добавлять пакеты подготовки, которые могут попытаться переопределить заблокированные политики.                         |
|     Политики/безопасность/AllowRemoveProvisioningPackage    |     Нет                            |     Запрещает всем пользователям удалять этот заблокированный пакет подготовки.                                                           |
|     Политики/система/AllowLocation                       |     Нет                            |     Предотвращает попытки устройства отслеживать данные о местоположении.                                                                        |
|     Политики/WiFi/AllowWiFi                             |     Нет                            |     Отключение Wi-Fi                                                                                                                 |

4. В разделе Параметры среды выполнения выберите **учетные записи/пользователи/имя пользователя: Holo и пароль.** 
    - Запишите пароль и при необходимости измените его.
5. Перейдите на UniversalAppInstall/UserContextApp и [Настройте бизнес-приложение](app-deploy-provisioning-package.md) , которое будет развертываться на этих устройствах.

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана, на котором показано, как добавить приложение в WCD.](images/offline-secure-sample-wcd-usercontextapp.png)

6. После завершения нажмите кнопку "экспорт" и выполняйте все запросы, пока не будет создан пакет подготовки.

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана: кнопка "экспорт" для этого пакета в WCD.](images/offline-secure-sample-wcd-export.png)


## Развертывание
1. Подключите HL2 к компьютеру с Windows 10 через USB-кабель.
1. Запуск инструмента "Дуга" и выбор **HoloLens 2**

   <img src=images/offline-secure-arc-1.png alt=arc1 width="577" height="322" />

1. На следующем экране выберите **ручной набор пакетов**.
   
   <img src=images/offline-secure-arc-2.png alt=arc2 width="577" height="322" />

1. Перейдите к ранее загруженному FFU файлу и нажмите кнопку **Открыть**.
1. На странице предупреждения нажмите кнопку **продолжить**.

   <img src=images/offline-secure-arc-3.png alt=arc3 width="577" height="322" />

1. Дождитесь завершения установки системы HoloLens 2 OS.
1. После завершения установки и загрузки устройства на компьютере перейдите в проводник и скопируйте ранее сохраненный файл PPKG в папку устройства.

   > [!div class="mx-imgBorder"]
   > ![PPKG файл на компьютере с Windows в проводнике.](images/offline-secure-file-explorer.png)

1. На HoloLens 2 нажмите приведенную ниже комбинированную кнопку, чтобы запустить пакет подготовки: нажмите **кнопку "** **уменьшить громкость** " и выберите один раз.
1. Вам будет предложено применить пакет подготовки, а затем нажмите кнопку **подтвердить** .
1. После завершения установки пакета подготовки нажмите кнопку **ОК**.
1. После этого вам будет предложено войти в устройство с помощью общей локальной учетной записи и пароля.

## Обслуживание
В этой конфигурации рекомендуется перезапустить описанную выше процедуру и перекрасить устройство с помощью инструмента ARC и применить новый PPKG, чтобы внести изменения в операционную систему и/или приложения.
