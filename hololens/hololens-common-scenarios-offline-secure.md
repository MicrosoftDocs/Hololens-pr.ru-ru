# Распространенные сценарии — автономный безопасный holoLens 2

## Обзор

В этом руководстве содержится руководство по использованию примера пакета подготовка, который блокирует HoloLens 2 для использования в безопасных средах со следующими ограничениями:
-   Отключать Wi-Fi.
-   Отключать BlueTooth.
-   Отключать микрофоны.
-   Предотвращает добавление или удаление пакетов подготовка.
-   Ни один пользователь не может включить какие-либо из вышеуказанных ограниченных компонентов.

## Подготовка

Установка компьютера с Windows 10
1. [Скачайте последний файл ОС HoloLens 2](https://aka.ms/hololens2download) непосредственно на компьютер. 
   1. Поддержка этой конфигурации включена в сборку 19041.1117 и выше.
1. Скачивание и установка средства Advanced Recovery Companion(ARC) из [Microsoft Store](https://www.microsoft.com/store/productId/9P74Z35SFRS8) на компьютер
1. Скачайте и установите последнюю версию конструктора конфигураций [Windows (WCD)](https://www.microsoft.com/p/windows-configuration-designer/9nblggh4tx22?activetab=pivot:overviewtab) из Microsoft Store на компьютер.
1. [Загрузите OfflineSecureHL2_Sample папку с файлами проекта](https://aka.ms/HoloLensDocs-SecureOfflineSample) для сборки PPKG.
1. Подготовьте [автономное бизнес-приложение для развертывания PPKG.](app-deploy-provisioning-package.md) 


## Настройка

Создание пакета безопасной настройки

1. Запустите средство WCD на компьютере.
1. Select **File -> Open project**.
  1. Перейдите к расположению ранее сохраненной OfflineSecureHL2_Sample и выберите: OfflineSecureHL2_Sample.icdproj.xml
1. Проект должен открыться, и теперь у вас должен быть список доступных настроек: 

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана: пакет конфигурации, открытый в WCD](images/offline-secure-sample-wcd.png)

Конфигурации, заданная в этом пакете предоставления:

|     Элемент                                                |     Параметр                       |     Описание                                                                                                                    |
|---------------------------------------------------------|-----------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
|     Учетные записи и пользователи                                    |     Локальный пароль & пользователя    |     Для таких автономных устройств необходимо установить одно имя пользователя и пароль, которые будут общими для всех пользователей устройства.          |
|     First Experience / HoloLens / SkipCalibration       |     True                          |     Пропускает калибровку только при начальной настройке устройства                                                                             |
|     First Experience / HoloLens / SkipTraining          |     True                          |     Пропускает обучение устройств во время начальной настройки устройства                                                                              |
|     First Experience / HoloLens / WiFi                  |     True                          |     Пропускает Wi-Fi конфигурацию во время начальной настройки устройства                                                                                 |
|     Policies/Connectivity/AllowBluetooth                |     Нет                            |     Отключает Bluetooth                                                                                                             |
|     Policies/Experience/AllowCortana                    |     Нет                            |     Отключает Кортану (чтобы устранить возможные проблемы, так как микрофоны отключены)                                          |
|     Policies/MixedReality/MicrophoneDisabled            |     Да                           |     Отключает микрофон                                                                                                            |
|     Policies/Privacy/LetAppsAccessLocation              |     Принудительное запрет                    |     Предотвращает попытки приложений получить доступ к данным о расположении (чтобы устранить потенциальные проблемы, так как отслеживание расположения отключено)    |
|     Policies/Privacy/LetAppsAccessMicrophone            |     Принудительное запрет                    |     Предотвращает попытки приложений получить доступ к микрофонам (чтобы устранить возможные проблемы, так как микрофоны отключены)           |
|     Policies/Security/AllowAddProvisioningPackage       |     Нет                            |     Предотвращает добавление пакетов предоставления, которые могут пытаться переопредить заблокированные политики.                         |
|     Policies/Security/AllowRemoveProvisioningPackage    |     Нет                            |     Предотвращает удаление этого заблокированного пакета подготовка.                                                           |
|     Policies/System/AllowLocation                       |     Нет                            |     Предотвращает попытки устройства отслеживать данные о расположении.                                                                        |
|     Policies/WiFi/AllowWiFi                             |     Нет                            |     Отключает Wi-Fi                                                                                                                 |

4. В области "Параметры времени работы" выберите **"Учетные записи/ Пользователи/ Имя пользователя: Holo / Пароль"** 
    - При желании заметьте пароль и сброс.
5. Перейдите к universalAppInstall / UserContextApp и настройте приложение [для](app-deploy-provisioning-package.md) развертывания на этих устройствах.

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана: место добавления приложения в WCD.](images/offline-secure-sample-wcd-usercontextapp.png)

6. После завершения выберите кнопку "Экспорт" и следуйте всем запросам, пока пакет подготовка не будет создан.

   > [!div class="mx-imgBorder"]
   > ![Снимок экрана с кнопкой "Экспорт" для этого пакета в WCD.](images/offline-secure-sample-wcd-export.png)


## Развертывание

1. Подключите HL2 к компьютеру с Windows 10 через USB-кабель.
1. Запустите средство ARC и выберите **HoloLens 2**

   <img src=images/offline-secure-arc-1.png alt=arc1 width="577" height="322" />

1. На следующем экране выберите выбор **пакета вручную.**
   
   <img src=images/offline-secure-arc-2.png alt=arc2 width="577" height="322" />

1. Перейдите к ранее загруженным FFU-файлу и выберите **"Открыть".**
1. На странице предупреждения выберите **"Продолжить"**.

   <img src=images/offline-secure-arc-3.png alt=arc3 width="577" height="322" />

1. Подождите, пока инструмент ARC завершит установку ОС HoloLens 2.
1. После завершения установки и загрузки устройства перейдите с компьютера в проводник и скопируйте ранее сохраненный файл PPKG в папку устройства.

   > [!div class="mx-imgBorder"]
   > ![Файл PPKG на компьютере в окне проводника.](images/offline-secure-file-explorer.png)

1. На HoloLens 2 нажмите следующую кнопку со combo **** для одновременного запуска пакета подготовка: коснитесь кнопки "Громкость вниз" и **"Кнопка питания".**
1. Вам будет предложено применить пакет предоставления, выберите **"Подтвердить"**
1. После завершения пакета подготовка выберите **"ОК".**
1. После этого вам будет предложено войти на устройство с помощью общей локальной учетной записи и пароля.

## Обслуживание

При такой конфигурации рекомендуется перезапустить процесс выше и перезапустить устройство с помощью инструмента ARC и применить новый PPKG для внесения любых обновлений в ОС и/или приложениях.
