# [HoloLens 2 Development Edition](#tab/developeredition)

## 1. Узнайте о выпуске Development Edition

> [!NOTE]
> Выпуск Development Edition не может быть развернут с коммерчески распределенным решением. Выпуск Development Edition может быть развернут при разработке и (или) тестировании приложений в коммерческой среде.  

Выпуск HoloLens 2 Development Edition включает holoLens 2, 500 долларов США в кредиты Azure, трехмесячную пробную версию Unity Pro и 3-месячную пробную версию подключаемого модуль piXYZ:

| Функция пакета | Описание |
|---|---|
|  [Unity](https://unity.com/) | Платформа для разработки трехd в режиме реального времени.   |
|  [Подключаемый модуль Pixyz для Unity](https://www.pixyz-software.com/plugin/) | Мост для подключения мира &amp; САИБ для разработки с миром разработки.   |
| [500 долларов США в кредитах Azure для служб Azure](https://azure.microsoft.com/resources/) | Кредиты Azure, включенные в Выпуск разработки HoloLens 2, — это быстрое начало создания и тестирования приложений смешанной реальности в Azure. |

Каждый выпуск HoloLens 2 имеет [](../hololens2-hardware.md#hololens-components) одинаковое оборудование, то же содержимое в поле и поставляется с тем же преду [установить программное обеспечение.](../hololens2-hardware.md#pre-installed-software)

## 2. Погагайте кредиты

### Unity
Вы получите уникальный код. Маршруты для активировать пробную версия Unity Pro:
1. Посетите [id.unity.com](http://id.unity.com/)
1. Во sign in with your Unity ID or create one
1. Click **Redeem a product code**
1. Введите код продукта и нажмите **кнопку "Активировать код продукта"**

Дополнительные сведения об [активации лицензии Unity.](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-) Обратитесь [за помощью в](https://support.unity3d.com/hc) службу поддержки Unity по мере необходимости.  

### Pixyz
Вы получите уникальный код. Инструкции по выкупу пробной Pixyz Plugin:
1. Посетите [id.unity.com](http://id.unity.com/)
1. Click **Redeem a product code**
1. Введите код подключаемой программы Pixyz и нажмите кнопку **"Активировать код продукта"**

Дополнительные сведения о том, как [активировать лицензию Pixyz.](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html) Обратитесь [в службу поддержки Pixyz за](https://www.pixyz-software.com/support/) помощью по мере необходимости.

### Azure
Вот некоторые распространенные [продукты Azure для смешанной реальности,](https://azure.microsoft.com/topic/mixed-reality/) за которые вы можете получить кредит Azure.
Как активировать кредиты Azure: чтобы активировать кредиты Azure на сумму 500 долларов США, активируйте промокод Azure Pass и активируйте подписку. [Подробнее.](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [Узнайте, как пространственные привязки Azure помогают создавать приложения, которые соединую, совместно используют и сохраняют трехdовую содержимого](https://azure.microsoft.com/services/spatial-anchors/)
- [Узнайте, как удаленная отрисовка Azure может отрисовки высококачественного интерактивного 3D-содержимого и потоковой передачи его на устройства в режиме реального времени](https://azure.microsoft.com/services/remote-rendering/)
