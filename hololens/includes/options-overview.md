# [Выпуск для версии для разработчиков HoloLens 2](#tab/developeredition)

## 1. сведения о выпуске для разработчиков

> [!NOTE] 
> Выпуск для разработки не разрешается развертывать вместе с коммерчески распространяемым решением. Выпуск для разработчиков разрешается развертывать в целях разработки и (или) тестирования приложений в коммерческих параметрах.  

Выпуск для архитектуры HoloLens 2 включает HoloLens 2, $500 в Azure кредиты, пробную версию Unity Pro на 3 месяца и пробную версию подключаемого модуля PiXYZ на 3 месяца.

| Пакетная функция | Описание |
|---|---|
|  [Unity](https://unity.com/) | Трехмерная платформа для разработки в реальном времени.   |
|  [Плагин Pixyz для Unity](https://www.pixyz-software.com/plugin/) | Мост для подсоединения инженеров по проектированию &amp; САПР в мире для разработки.   |
| [$500 в Azure кредиты для служб Azure](https://azure.microsoft.com/resources/) | Кредитные кредиты Azure, включенные в выпуск для разработчиков HoloLens 2, помогут вам быстро освоить создание и тестирование приложений смешанной реальности в Azure. |

Каждый выпуск HoloLens 2 имеет одинаковое оборудование, то же самое [содержимое в поле](../hololens2-hardware.md#hololens-components) и поставляется с одним и тем же [предварительно установленным программным обеспечением](../hololens2-hardware.md#pre-installed-software).

## 2. Попросите своих кредитов

### Unity
Ты получит уникальный код. Инструкции по активации пробной версии Unity Pro: 
1. Посетить [ID.Unity.com](http://id.unity.com/) 
1. Выполните вход с помощью идентификатора Unity или создайте его.
1. Щелкните " **Активировать код продукта** " 
1. Введите код продукта и щелкните " **Активировать код продукта** "

Дополнительные сведения о [том, как активировать лицензию Unity](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-). Обратитесь в [службу поддержки Unity](https://support.unity3d.com/hc) , чтобы получить помощь по мере необходимости.  

### Pixyz
Ты получит уникальный код. Инструкции по активации пробной версии подключаемого модуля Pixyz: 
1. Посетить [ID.Unity.com](http://id.unity.com/)
1. Щелкните " **Активировать код продукта** "
1. Введите код продукта подключаемого модуля Pixyz и нажмите кнопку **Активировать код продукта**

Дополнительные сведения о том, как [активировать лицензию на Pixyz.](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html) Обратитесь в [службу поддержки Pixyz](https://www.pixyz-software.com/support/) , чтобы получить помощь по своему усмотрению. 

### Azure
Ниже приведены некоторые распространенные [продукты Azure для смешанной реальности](https://azure.microsoft.com/topic/mixed-reality/) , на которые вы можете пожелать Вам воспользоваться вашим кредитом в Azure.
Как использовать кредитные кредиты в Azure: чтобы активировать ваши кредиты в $500 Azure, используйте рекламныйный код Pass для Azure и активируйте подписку. [Подробнее.](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [Сведения о том, как с помощью пространственных якорей Azure можно создавать приложения, которые сопоставлены, совместно использовать и хранят трехмерные материалы.](https://azure.microsoft.com/services/spatial-anchors/)
- [Сведения о том, как удаленная визуализация Azure может визуализировать высококачественное, интерактивное трехмерное содержимое и передавать их в реальном времени в потоковом режиме.](https://azure.microsoft.com/services/remote-rendering/)
