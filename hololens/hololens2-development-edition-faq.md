# <a name="hololens-2-development-edition-faq"></a>HoloLens 2 Development Edition: вопросы и ответы

## <a name="general-questions"></a>Общие вопросы

### <a name="what-do-i-get-with-the-hololens-2-development-edition"></a>Что можно получить в выпуске HoloLens 2 Development Edition?

В выпуск HoloLens 2 Development Edition входит HoloLens 2, $500 в кредитах Azure, пробная версия Unity Pro на 3 месяца и пробная версия 3-месячного подключаемого модуля Пиксиз.

### <a name="what-is-the-difference-between-hololens-2-and-hololens-2-development-edition"></a>В чем разница между HoloLens 2 и HoloLens 2 Development Edition?

Выпуск HoloLens 2 Development Edition можно использовать только для создания и тестирования внутренних корпоративных приложений, которые будут использоваться с HoloLens 2 (коммерческий выпуск) в рабочей среде. В рамках приобретения выпуска Development Корпорация Майкрософт может, по своему усмотрению, предоставлять вам определенные права, идентифицированные в процессе извлечения. Эти права включают право на получение определенных кредитов на скидку или пробный доступ к определенным службам Майкрософт или сторонним средствам или инструментам в течение ограниченного периода. [Дополнительные сведения](hololens2-options-dev-edition.md)

### <a name="can-any-edition-of-hololens-2-be-used-to-develop"></a>Можно ли использовать любой выпуск HoloLens 2 для разработки?

Да. Для разработки приложений можно использовать любой выпуск HoloLens 2.

### <a name="how-and-will-i-receive-my-500-in-azure-credits-3-month-unity-pro-trial-and-3-month-pixyz-plugin-trial"></a>Как и я получаю $500 в кредитах Azure, в течение 3-месячного пробного выпуска и в течение 3-месячного подключаемого модуля Пиксиз?

Ваши кредиты на скидку или пробный доступ будут отправляться по электронной почте, используемой при покупке в течение двух рабочих дней с момента отгрузки продукта.

### <a name="can-i-upgrade-my-hololens-2-device-only-device-to-the-hololens-2-development-edition-to-get-the-trial-credits"></a>Могу ли я "Обновить" устройство HoloLens 2 (только устройство) до выпуска HoloLens 2, чтобы получить пробные кредиты?

Нет, мы не предлагаем планы перемещения устройств. Невозможно выполнить перенос из любого выпуска в любой другой выпуск.

### <a name="what-is-unity-pro"></a>Что такое Unity Pro?

Unity Pro — это трехмерная платформа в режиме реального времени для малых и средних studiosов и компаний. Оно предназначено для групп, которым не нужны технические и нетехнические навыки. [Дополнительные сведения](https://store.unity.com/products/unity-pro)

### <a name="what-is-pixyz-plugin"></a>Что такое подключаемый модуль Пиксиз?

Подключаемый модуль Пиксиз считывает и преобразует почти любой формат, оптимизирует и размещает его в Unity, а также предоставляет готовые ресурсы для приложений в реальном времени. [Дополнительные сведения](https://unity.com/products/pixyz)

### <a name="is-financing-for-the-hololens-2-development-edition--available"></a>Является ли финансирование доступным для выпуска HoloLens 2?

Да, [финансирования для HoloLens 2 Development Edition](https://ms.liftforward.com/hololens?product=hololens-development-edition) теперь доступен только в США. 

### <a name="who-do-i-contact-with-questions-about-my-hololens-2-development-edition-entitlements"></a>Как связаться с вопросами о моих правах на выпуск HoloLens 2 Development Edition?

С вопросами, связанными с назначениями, обратитесь к специалисту по продажам по адресу 866-425-4709, от ПН до пт, 06:00 до 18:00, PST.

### <a name="can-my-company-buy-multiple-hololens-2-development-edition-devices"></a>Может ли моя компания купить несколько устройств с версией HoloLens 2 Development Edition? 

Это устройство предназначено для разработчиков, создающих решения смешанной реальности. Мы будем ограничивать одну покупку на каждую клиентскую транзакцию. Если вашей компании требуется приобрести несколько устройств выпуска для разработки, вам потребуется отдельная транзакция для приобретения устройств HoloLens 2 Development Edition от имени компании. 

### <a name="can-i-show-my-hololens-2-development-edition-device-to-my-customers"></a>Можно ли показывать мое устройство HoloLens 2 Development Edition пользователям? 

Да, это устройство предназначено для использования при создании прототипов и тестировании решений смешанной реальности.

### <a name="where-can-i-buy-the-hololens-2-development-edition"></a>Где можно купить выпуск HoloLens 2 Development Edition? 

Выпуск HoloLens 2 Development Edition реализован исключительно на Microsoft Store.  

### <a name="can-i-return-my-hololens-2-development-edition"></a>Можно ли вернуть мой выпуск HoloLens 2 Development Edition?  

Все продажи являются окончательными, и в версии HoloLens 2 Development Edition нет доступных возвратов.

### <a name="does-developer-edition-have-different-hardware-requirements-that-differ-from-the-hololens-2"></a>Существуют ли в версии Developer разные требования к оборудованию, отличные от HoloLens 2?

Нет, это устройство совпадает с перспективой "требования к оборудованию". Выпуск Development Edition разрешается развертывать в среде разработки и (или) тестирования приложений в коммерческих параметрах. 

### <a name="how-can-i-use-mixed-reality-toolkit--unity-for-developing-on-my-hololens-2-development-edition"></a>Как можно использовать набор средств для смешанной реальности — Unity для разработки в моем выпуске HoloLens 2 Development Edition?

MRTK-Unity — это проект, управляемый Майкрософт, который предоставляет набор компонентов и функций для ускорения кроссплатформенной разработки приложений смешанной реальности в Unity. [Дополнительные сведения](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### <a name="how-can-i-use-mixed-reality-toolkit--unreal-uxt-for-developing-on-my-hololens-2-development-edition"></a>Как можно использовать набор средств для смешанной реальности — Нереал УКСТ для разработки в моем выпуске HoloLens 2 Development Edition?

MRTK-Unreal — это первый компонент средств UX, который должен быть выпущен и сейчас поддерживается только в HoloLens 2. Подключаемый модуль компонента включает код, схемы и примеры ресурсов распространенных функций пользовательского интерфейса для имитации ввода, субъектов взаимодействия с рукой, компонентов кнопок для нажатия, компонентов манипуляторов и компонентов следования поведению. [Дополнительные сведения](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### <a name="are-os-releases-for-the-hololens-2-development-edition-the-same-as-hololens-2"></a>Существуют ли выпуски ОС для выпуска HoloLens 2, такие же, как HoloLens 2?

Да.

### <a name="can-i-run-released-sideload-or-store-applications-on-my-hololens-2-development-edition-device"></a>Можно ли запускать выпущенные (загружать неопубликованные или хранящие) приложения на устройстве с версией HoloLens 2 Development Edition?

Если вы абидинг с условиями выпуска HoloLens 2 Development Edition, да, вы можете запустить любое выпущенное приложение в выпуске Developer Edition. Мы не будем ограничивать использование загруженные неопубликованные или хранимых приложений для этого SKU.

### <a name="where-can-i-learn-more-about-the-hololens-2-development-edition"></a>Где можно получить дополнительные сведения о выпуске HoloLens 2 Development Edition?

Посетите страницу документации по версии для среды разработки HoloLens 2. [Дополнительные сведения](https://docs.microsoft.com/hololens/hololens2-options?tabs=device)

## <a name="azure"></a>Azure

### <a name="how-can-i-redeem-my-500-azure-credit"></a>Как активировать $500 Azure для кредита?

Чтобы активировать $500 Azure на счете, активируйте Azure Pass код Акционная и активизируйте подписку. [Дополнительные сведения](https://www.microsoftazurepass.com/Home/HowTo?Length=5)

Ознакомьтесь с [соглашением о подписке](https://azure.microsoft.com/support/legal/subscription-agreement/?country=US) Azure Pass и [сведениями о предложении](https://azure.microsoft.com/en-us/offers/azure-pass/).

### <a name="when-does-my-500-azure-credit-expire"></a>Когда истекает срок действия моего кредита Azure $500?

Срок действия вашего кредита Azure $500 истечет через 90 дней после активации кода ключа Azure.

### <a name="what-can-i-do-with-my-azure-credits"></a>Что можно делать с кредитами Azure?

С вашими кредитами Azure вы можете начать разработку с помощью Microsoft&#39;в постоянно развертываемом наборе облачных служб. Кредиты Azure, входящие в состав выпуска HoloLens 2 Development Edition, позволяют пользователям быстро приступить к созданию и тестированию приложений смешанной реальности в Azure. Подробные сведения о создании в Azure см. на странице [ресурсов Azure](https://azure.microsoft.com/en-us/resources/) .

Корпорация Майкрософт предлагает набор служб смешанной реальности, позволяющих разработчикам создавать и тестировать решения смешанной реальности. [Дополнительные сведения](https://docs.microsoft.com/windows/mixed-reality/develop/mixed-reality-cloud-services)
