# Настройка HoloLens с помощью пакета подготовки

[Подготовка Windows](https://docs.microsoft.com/windows/configuration/provisioning-packages/provisioning-packages) позволяет ИТ-администраторам легко настраивать устройства пользователей, не требующая передачи. Конструктор конфигураций Windows — это средство для настройки изображений и параметров среды выполнения, которые затем встроены в пакеты подготовки.

Ниже перечислены некоторые конфигурации HoloLens, которые можно применить в пакете подготовки.

- Переход на Windows Holographic для [бизнеса здесь](hololens1-upgrade-enterprise.md)
- Настройка локальной учетной записи
- Настройка подключения к сети Wi-Fi
- Применить сертификаты к устройству
- Включение режима разработчика
- Здесь можно настроить режим киоска (подробные инструкции по настройке работы в киоске) приведены [здесь.](hololens-kiosk.md#use-a-provisioning-package-to-set-up-a-single-app-or-multi-app-kiosk)

## Мастер подготовки HoloLens

Мастер HoloLens поможет настроить следующие параметры в пакете подготовки:

- Обновление до корпоративного выдачи

    > [!NOTE]
    > Это можно использовать только для устройств HoloLens 1 гента. Параметры в пакете подготовки применяются только в том случае, если пакет подготовки включает лицензию на выпуск Windows Holographic для бизнеса или если устройство уже было [обновлено до Windows Holographic для бизнеса.](hololens1-upgrade-enterprise.md)

- Настройка первого взаимодействия HoloLens (OOBE)
- Настройка сети Wi-Fi
- Регистрация устройства в Azure Active Directory или создание локальной учетной записи
- Добавление сертификатов
- Включение режима разработчика
- Настройка режима киоск. (Здесь приведены подробные инструкции по включению режима киооек можно найти [здесь).](hololens-kiosk.md#use-a-provisioning-package-to-set-up-a-single-app-or-multi-app-kiosk)

> [!WARNING]
> Необходимо запустить конструктор конфигураций Windows в Windows 10 для настройки регистрации Azure Active Directory с помощью любого из мастеров.

Пакеты подготовки могут включать инструкции по управлению и политиками, настраиваемым сетевым подключениям и политикам и т. д.

> [!TIP]
> Используйте мастер настольных компьютеров, чтобы создать пакет с общими параметрами, затем переключитесь в расширенный редактор, чтобы добавить другие параметры, приложения, политики и т. д.

### 2. Создание пакета подготовки

Для создания пакета подготовки используйте конструктор конфигураций Windows.

1. Откройте конструктор конфигураций Windows (по умолчанию %windir%\Program Files (x86)\Windows Kits\10\Assessment and Deployment Kit\Imaging and Configuration Designer\x86\ICD.exe).

2. Выберите **устройства HoloLens.**

   ![Параметры запуска ICD](images/icd-create-options-1703.png)

3. Введите имя проекта и нажмите **кнопку "Готово".**

4. Прочитайте инструкции на странице **"Приступая к** работе" и нажмите кнопку "Далее". **Next** Ниже описаны ссылки на страницы для подготовки настольных компьютеров.
  
> [!IMPORTANT]
> При создании пакета подготовки вы можете включить конфиденциальную информацию в файлы проекта и в файл пакета подготовки (PPKG). Несмотря на то что у вас есть возможность шифрования PPKG-файла, файлы проекта не шифруются. Файлы проекта следует хранить в надежном месте, если они больше не требуются, их необходимо удалять.

### 3. Создание пакета подготовки для HoloLens с помощью расширенной подготовки

> [!NOTE]
> Пакет подготовки, созданный в **расширенной подготовке,** не требуется включать лицензию на Windows Holographic for Business, чтобы успешно применять HoloLens к HoloLens (1 год). [Дополнительные сведения о Windows Holoграфические данные для HoloLens (1 гезенды) см. в Windows HoloLens (1 географическое).](hololens1-upgrade-enterprise.md)

1. На начальной странице конструктора конфигураций Windows нажмите **Расширенная подготовка**.
2. В окне **Введите сведения о проекте** укажите имя и расположение своего проекта. При желании вы также можете ввести краткое описание своего проекта.

3. Выберите **Далее**.

4. В **окне "Выбор параметров для просмотра и настройки** окна" щелкните **"Windows 10 Holographic",** а затем нажмите **кнопку "Далее".**

5. Нажмите **кнопку "Готово".**

6. Разверните **параметры среды** выполнения и настройте пакет с помощью параметров, описанных [далее в этой статье.](#what-you-can-configure)

    > [!IMPORTANT]
    > (Только для Windows 10 версии 1607) При создании локальной учетной записи в пакете подготовки вам необходимо изменить пароль с помощью приложения **"Настройки".** Если пароль не меняется в течение этого периода, учетная запись может быть заблокирована и стать недоступной для входа. Если учетная запись пользователя заблокирована, необходимо [выполнить полное восстановление устройства](https://developer.microsoft.com/windows/mixed-reality/reset_or_recover_your_hololens#perform_a_full_device_recovery).

7. Нажмите **кнопку**  >  **"Сохранить**файл".

8. Прочтите предупреждение о том, что файлы проектов могут содержать конфиденциальную информацию, и нажмите кнопку **"ОК".**

    > [!IMPORTANT]
    > При создании пакета подготовки вы можете включить конфиденциальную информацию в файлы проекта и в файл пакета подготовки (PPKG). Несмотря на то что у вас есть возможность шифрования PPKG-файла, файлы проекта не шифруются. Файлы проекта следует хранить в надежном месте, если они больше не требуются, их необходимо удалять.
    
9. Выберите **Export**  >  **"Экспорт".**

10. Изменить **владельца** **на ИТ-администратор.** Этот параметр задает приоритет пакета подготовки, чем отсутствует пакеты подготовки к этому устройству из других источников. Выберите **Далее**.

11. Задайте значение параметра **Версия пакета**.

    > [!TIP]
    > Можно внести изменения в существующие пакеты и изменить номер версии для обновления ранее примененных пакетов.

12. В разделе **"Выберите сведения о безопасности" для пакета подготовки**нажмите кнопку **"Далее".**

    > [!WARNING]
    > Если зашифровать пакет подготовки, при подготовке устройства HoloLens возникнет ошибка.  

13. Нажмите **кнопку** "Далее" для указания места, в котором должен находиться пакет подготовки. По умолчанию конструктор конфигураций Windows использует в качестве расположения выходных данных папку проекта.

    При необходимости вы можете **изменить** местоположение вывода по умолчанию.

14. Выберите **Далее**.

15. Выберите **"Сборка"** для создания пакета. Сведения о проекте отображаются на странице сборки, а индикатор выполнения указывает состояние сборки.

16. После завершения сборки нажмите **кнопку "Готово".**

## Применение пакета подготовки к HoloLens во время настройки

Устройства HoloLens 2 в сборке [19041.1103](hololens-release-notes.md#windows-holographic-version-2004) или более поздней версии могут использовать USB-настольный диск для применения пакета подготовки. Просто скопируйте PPKG-файл в корню юридического диска. Пакеты подготовки будут применяться только в том случае, если они находятся в корневом USB-настольных устройствах. Несколько пакетов подготовки будет последовательно применяться к нескольким пакетам подготовки.

1. Используйте USB-кабель для подключения устройства к ПК (или USB-накоплении HoloLens 2 как говорилось выше), а затем запустите устройство. Не продолжайте предыдущую **страницу** максимального интерфейса OOBE.   
    - В HoloLens (1 года) эта страница содержит синюю рамку. 
    - В HoloLens 2 эта страница содержит громкость.

2. Одновременно и кратковременно нажмите и отпустите кнопки **уменьшения громкости** и **питания**. 

3. Устройство HoloLens отображается в проводнике на компьютере в виде устройства.

4. В проводнике перетащите пакет подготовки (.ppkg) в раздел хранения данных на устройстве.

5. Коротко ее нажатия и **Power** одновременно ее выпускать одновременно на странице многократного перетаскивания OOBE. **Volume Down** **First interactable moment**

6. Устройство просит вас, доверяете ли пакету, и хотите применить его. Подтвердите, что доверяете пакету.

7. Вы увидите, успешно ли прошло применение пакета. Если произойдет сбой, можно исправить пакет и повторить попытку. Если применение выполнено успешно, продолжите процедуру запуска при первом включении устройства.

> [!NOTE]
> Если устройство было приобретено до августа 2016 г., для применения пакета на устройстве необходимо войти в систему с помощью учетной записи Майкрософт, установить последнее обновление операционной системы и затем сбросить операционную систему, чтобы применить этот пакет.

## Применение пакета подготовки к HoloLens после установки

> [!NOTE]
> Эти действия применимы только к Windows 10 версии 1809.

На компьютере выполните указанные ниже действия.
1. Создайте пакет подготовки для [HoloLens с помощью мастера HoloLens.](hololens-provisioning.md)
2. Подключите устройство HoloLens к компьютеру с помощью USB-кабеля. Устройство HoloLens отображается в проводнике на компьютере в виде устройства.
3. Перетащите пакет подготовки в папку "Документы" на устройстве HoloLens.

На устройстве HoloLens выполните следующие действия:
1. Перейдите к **параметрам учетной**  >  **записи**  >  **рабочей или учебной учетной записи.** 
2. В **области "Связанные параметры"** **выберите "Добавить или удалить пакет подготовки".**
3. На следующей странице нажмите **кнопку "Добавить пакет",** чтобы запустить средство выбора файлов, и выберите подготовку. Если папка пустая, убедитесь, что вывыберете **это устройство и** щелкните **"Документы".**

После применения пакета она отобразится в списке **установленных пакетов.** Чтобы просмотреть сведения о пакете или удалить пакет с устройства, выберите его в списке.

## Что можно настроить

Пакеты подготовки используют поставщики услуг конфигурации (CSP). Если вы не знакомы с понятием «поставщик службы конфигурации» (CSP), сначала ознакомьтесь с разделом [Вводные сведения о поставщиках службы конфигурации (CSP) для ИТ-профессионалов](https://docs.microsoft.com/windows/configuration/provisioning-packages/how-it-pros-can-use-configuration-service-providers).

В конструкторе конфигураций Windows при создании пакета подготовки для Windows Holographic параметры в разделе **Доступные настройки** основаны на [поставщиках CSP, которые поддерживаются в среде Windows Holographic](https://docs.microsoft.com/windows/client-management/mdm/configuration-service-provider-reference#csps-supported-in-hololens-devices). В следующей таблице рассматриваются параметры, которые можно настроить для HoloLens.

![Общие параметры среды выполнения для HoloLens](images/icd-settings.png)

| Параметр | Описание |
| --- | --- |
| **Сертификаты** | Развертывание сертификата на HoloLens.  |
| **ConnectivityProfiles** | Развертывание профиля Wi-Fi на HoloLens.   |
| **EditionUpgrade** | [Обновление до Windows Holographic for Business.](hololens1-upgrade-enterprise.md)  |
| **Политики** | Разрешение или запрет режима разработчика на устройстве HoloLens. [Политики, поддерживаемые Windows Holographic for Business](https://docs.microsoft.com/windows/client-management/mdm/policy-configuration-service-provider#hololenspolicies) |

> [!NOTE]
> В настоящее время HoloLens не поддерживает установку приложений **(UniversalAppInstall)** с помощью пакета подготовки.
