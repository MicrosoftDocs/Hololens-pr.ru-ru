### Знакомство с новым Microsoft Edge

![Анимация устаревшего логотипа Microsoft Edge до нового логотипа Microsoft Edge](images/new-edge.gif)

Новый microsoft Edge принимает проект с открытым исходным кодом [Chromium,](https://blogs.windows.com/windowsexperience/2018/12/06/microsoft-edge-making-the-web-better-through-more-open-source-collaboration/) чтобы улучшить совместимость для клиентов и менее фрагментацию веб-сайта для веб-разработчиков.

С помощью этой предварительной версии новый Microsoft Edge доступен пользователям HoloLens 2 в первый раз! Хотя новый Microsoft Edge в конечном итоге заменит устаревший Microsoft Edge на HoloLens 2, оба браузера в настоящее время доступны для пользователей программы программы. Поделитесь своим мнением и ошибками с нашей командой с помощью функции **отправки отзывов** в новом Microsoft Edge или в [Центре отзывов.](hololens-feedback.md)

![Новый снимок экрана Microsoft Edge](images/new-edge-ui.png)

#### Запуск нового Microsoft Edge

Для программы insiders доступны две версии Microsoft Edge: новый значок Microsoft Edge (представленный синим и зеленым значком значка ).) и устаревший Microsoft Edge (представленный белым значком ![ ](images/new_edge_logo.png) "e"). Новый Microsoft Edge закреплен в меню "Пуск" и автоматически запускается при активации веб-ссылки. Если вы хотите вернуться к использованию Устаревшего Браузера Microsoft Edge в качестве веб-браузера по умолчанию, см. инструкции ниже для сброса приложений [по умолчанию.](#default-app-picker)

> [!NOTE]
> При первом запуске нового Microsoft Edge на HoloLens 2 параметры и данные будут импортироваться из устаревшей службы Microsoft Edge. Если вы продолжите использовать устаревший Microsoft Edge после запуска нового Microsoft Edge, эти новые данные не будут синхронизироваться с новым Microsoft Edge из устаревшей службы Microsoft Edge.

#### Настройка параметров политики для нового Microsoft Edge

Новый Microsoft Edge предлагает ИТ-администраторам гораздо более широкий набор политик браузера на HoloLens 2, чем ранее был доступен в Устаревшем Microsoft Edge.

Вот некоторые полезные ресурсы для получения дополнительных знаний об управлении настройками политики для нового Microsoft Edge:

- [Настройка параметров политики Microsoft Edge с помощью Microsoft Intune](https://docs.microsoft.com/deployedge/configure-edge-with-intune)
- [Сопоставление политик устаревшей версии Microsoft Edge и Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge)
- [Сопоставление политик Google Chrome и Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-chrome-to-newedge)
- Полная [документация по Microsoft Edge Enterprise](https://docs.microsoft.com/deployedge/)

> [!IMPORTANT]
> Из-за большого объема политик браузера, поддерживаемых новым Microsoft Edge, наша команда не может гарантировать, что каждая новая политика работает на HoloLens 2. Однако мы протестировали и проверили, что новый эквивалент Microsoft Edge для каждой устаревшей политики Microsoft Edge, ранее поддерживаемой на HoloLens 2, работает ожидаемым образом. См. сопоставление политик Microsoft Edge Legacy и [Microsoft Edge,](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge) чтобы найти новый эквивалент Microsoft Edge для каждой устаревшей политики браузера Microsoft Edge, которая использовался с HoloLens 2.
>
> Существует по крайней мере две новые политики Microsoft Edge, которые не *будут* работать с HoloLens 2:
> - EnterpriseModeSiteList
> - EnterpriseSiteListServiceURL

#### Чего ожидать от нового Microsoft Edge на HoloLens 2

Так как новый Microsoft Edge — это приложение Win32 с новым слоем адаптеров UWP, позволяющим работать только на устройствах UWP, таких как HoloLens 2, некоторые функции могут быть недоступны сразу. В ближайшие месяцы мы будем поддерживать новые сценарии и функции, поэтому ознакомьтесь с этой информацией в этом пространстве.

**Сценарии и функции, которые должны работать:**
- Первый запуск, вход в профиль и синхронизация
- Веб-сайты должны отрисовки и вести себя ожидаемым образом
- Большинство функций браузера (избранное, история и т. д.) должны работать ожидаемым образом.
- Темный режим
- Установка веб-приложений на устройство
- Установка расширений (сообщите нам, если вы используете какие-либо расширения, которые не работают должным образом на HoloLens 2)
- Просмотр и разметка PDF-файла
- Пространственный звук из одного окна браузера
- Автоматическое и ручное обновление браузера
- Сохранение PDF-файла из меню "Печать" (с помощью параметра "Сохранить в ФОРМАТ PDF")

**В ближайшее время будут представлены сценарии и функции:**
- WebXR и расширение viewer 360
- Восстановление содержимого для корректного окна при просмотре по нескольким окнам, размещенным в вашей среде

**Сценарии и функции не должны работать:**
- Пространственный звук из нескольких окон с одновременными аудиопотоками
- "See it, say it" (См.
- Вывод на печать

**Известные проблемы браузера:**
- Сброс устройства удалит новый Microsoft Edge
- Предварительный просмотр лубки на голографической клавиатуре показывает неправильное содержимое

Существует несколько методов установки каналов программы insider Microsoft Edge на HoloLens 2:

  1. [Запустится установщик](app-deploy-app-installer.md) приложений.

  1. После успешной установки microsoft Edge beta, Dev или Canary будет **** находиться в качестве отдельной записи в списке "Все приложения" меню "Пуск".

**Установка с компьютера с помощью [](https://docs.microsoft.com/windows/mixed-reality/develop/platform-capabilities-and-apis/using-the-windows-device-portal#setting-up-hololens-to-use-windows-device-portal) портала устройств с Windows (требуется включить режим разработчика на Устройстве HoloLens 2)**
  1. На компьютере посетите страницу загрузки [edge insider](https://www.microsoftedgeinsider.com/download).
  1. Выберите **кнопку со** стрелкой вниз рядом с кнопкой "Скачать для Windows 10" для канала edge Insider Channel, который вы хотите установить.

### WebXR и 360 Viewer

*Добавлено в сборку 20289.1000 для windows Insider*

Новый Microsoft Edge включает поддержку WebXR, который является новым стандартом для создания иммерсивных веб-функций (заменяющих WebVR). Многие иммерсивные веб-приложения были разработаны с учетом виртуальной реальности (они заменяют поле зрения виртуальной средой), но они также поддерживаются HoloLens 2. Стандарт WebXR также включает иммерсивные веб-функции дополненной и смешанной реальности, которые используют физическую среду. Так как разработчики проводят больше времени с WebXR, мы ожидаем, что пользователи HoloLens 2 смогут испытать новые иммерсивные эффекты дополненной и смешанной реальности!

Расширение 360 Viewer построено на webXR и автоматически устанавливается вместе с новым Microsoft Edge на HoloLens 2. Это веб-расширение дает возможность погрузиться в 360-градусные видео. YouTube предлагает самый большой выбор из 360 видео, поэтому мы рекомендуем начать с этого.

#### Использование WebXR

1. Перейдите на веб-сайт с поддержкой WebXR.
1. На веб-сайте выберите кнопку "Ввести **виртуальной** реальности". Расположение и визуальное представление этой кнопки могут отличаться для каждого веб-сайта, но могут выглядеть примерно так:

    ![Пример кнопки ВВОД ВИРТУАЛЬНОЙ РЕАЛЬНОСТИ](images/75px-enter-vr.png)

1. При первой попытке запуска webXR в определенном домене браузер запросит согласие на ввод иммерсивного представления, выберите **"Разрешить".**
1. Используйте [жесты HoloLens 2](hololens2-basic-usage.md#the-hand-tracking-frame) для управления опытом.
1. Если в этом случае нет кнопки **выхода,** используйте жест "Начните", чтобы вернуться домой. [](hololens2-basic-usage.md#start-gesture)

**Рекомендуемые примеры WebXR**
- 360 Viewer (см. следующий раздел)
- [Динозавры XR](https://www.xrdinosaurs.com/)
- [Barстовский экспресс](https://constructarca.de/game/barista-express/)
- [WebXR Paint](https://threejs.org/examples/webxr_vr_paint.html)

#### Использование 360 Viewer

1. Перейдите к 360-градусной видеозаписи на YouTube.
1. В видеокадре выберите кнопку гарнитуры смешанной реальности:

    ![Кнопка для активации 360 Viewer](images/enter-360-viewer.jpg)

1. При первой попытке запуска 360 Viewer в определенном домене браузер запросит согласие на ввод иммерсивного представления. Выберите **"Разрешить".**
1. [Касание по](hololens2-basic-usage.md#select-using-air-tap) осям для воспроизведения элементов управления воспроизведением. Используйте [лучи](hololens2-basic-usage.md#select-using-air-tap) рук и касание, чтобы играть/приостанавличивать, пропускать вперед и назад, выключать и выключать субтитры, а также останавливать впечатление (что выходит из иммерсивного представления). Элементы управления воспроизведением исчезнут через несколько секунд бездействия.

#### Известные проблемы WebXR и 360 просмотров
- В средах WebXR голограммы могут сдвигаться или наклоняться при наклоне головы или при движении вокруг среды.
- В зависимости от сложности работы с WebXR частота кадров может падать или перегреться.
- В WebXR пока недоступны скакумулирование ручного соединения.
- При выходе из webXR или просмотра 360 может потребоваться 30 секунд или больше, чтобы голограммы в домашней смешанной реальности появились снова.
- 360 видео с других веб-сайтов, кроме YouTube, могут работать не так, как ожидалось.
- Если 360 видео не входят в иммерсивное представление (или кнопка гарнитуры смешанной реальности не появляется), попробуйте обутить страницу.
- Субтитры пока не видны в 360 Viewer на HoloLens 2.
- Приоплата видео в 360 Viewer останавливает отрисовку видео (но нажатие кнопки воспроизведения правильно возобновляет воспроизведение).
- Кнопка "Следующее видео" в 360 Viewer в настоящее время не работает.
- Вы можете играть в двухмерные видео в иммерсивном режиме "зал", но частота кадров будет меньше 30 кадров в кадр/с.

### Приложение "Новые параметры"

В этом выпуске мы представляем новую версию приложения "Параметры". Новое приложение "Параметры" включает новые функции и расширенные параметры HoloLens 2 в следующих областях: "Звук", "Power & спящий режим", "Сетевой & Интернет", "Приложения", "Учетные записи", "Простота доступа" и другие.

> [!NOTE]
> Так как новое приложение "Параметры" отличается от устаревшего приложения "Параметры", все окна параметров, ранее размещенные в среде, будут удалены после обновления.

![Домашняя страницы приложения "Новые параметры"](images/new-settings-app.png)

**Новые функции и параметры**
- Поиск параметров: поиск параметров на домашней странице параметров с помощью ключевых слов или имени параметра.
- Системный > звук:
  - Звуковые устройства ввода и вывода: независимо выберите звуковые устройства ввода и вывода (например, прослушивание звука через наушники Bluetooth наушников или использование микрофона USB-C для ввода звука). 
    > [!NOTE]
    > Bluetooth микрофоны не поддерживаются HoloLens 2.
  - Громкость приложения: независимо отрегулировать объем каждого приложения.
- Системные > power & спящий режим: выберите, когда устройство должно перейти в спящий режим после периода бездействия.
- Системная > батареи: вручную включите режим экономии заряда или установите пороговое значение заряда батареи, при котором режим экономии заряда включается автоматически.
- Устройства > USB: по умолчанию можно отключить USB-подключения.
- Сетевые & Интернет:
  - Адаптеры Ethernet USB-C теперь будут отображаться в сетевом & Интернете.
  - Теперь доступны параметры адаптеров Ethernet USB-C, включая ЕГО IP-адрес.
  - Теперь можно включить режим "в самолете" на HoloLens 2.
- Приложения: вы можете сбросить приложения по умолчанию, используемые для типов файлов и ссылок. Дополнительные [сведения см.](#default-app-picker) в стандартной области "Выбор приложений".
- Учетные записи > других пользователей: владельцы устройств могут добавлять пользователей, обновлять стандартных пользователей до владельцев устройств, понизить уровень владельцев устройств до обычных пользователей и удалить пользователей.
- Простота доступа: изменение размера текста и некоторых визуальных эффектов.

**Известные проблемы**
- Ранее размещенные окна параметров будут удалены (см. примечание выше).
- При посещении страницы уведомлений может возникнуть сбой приложения "Параметры" (исследование).
- Страница Ethernet в настоящее время не появится (скоро она будет исправлена).
- Вы больше не сможете переименовать устройство с помощью приложения "Параметры" (ИТ-администраторы могут использовать пакеты подготовка или MDM для переименования устройств).
- Использование батареи для нового Microsoft Edge может быть не точным из-за его природы как настольного приложения Win32, поддерживаемого слоем адаптеров UWP (скоро исправление не ожидается).

### Прокрутка для введите

Некоторые клиенты быстрее "введите" виртуальные клавиатуры, протирая форму слова, которое они намерены ввести, и мы просматриваем эту функцию для голографической клавиатуры. Вы можете провести пальцем по одному слову, передав кончик пальца по плоскости голографической клавиатуры, прокрутирав форму слова, а затем выведя кончик пальца из плоскости клавиатуры. Вы можете прокрутки слов, не нажимая пробел, удалив палец от клавиатуры между словами. Вы будете знать, что эта функция работает, если вы видите след прокрутки после движения пальца на клавиатуре.

Обратите внимание, что эта функция может быть сложной для использования и хозяина из-за природы голографической клавиатуры, где вы не будете противяться пальцу (в отличие от дисплея мобильного телефона). Мы оцениваем эту функцию для общедоступных выпусков, поэтому ваш отзыв важен; Независимо от того, полезна ли функция или у вас есть обратная связь по отзывам, сообщите нам об этом через [Центр отзывов.](hololens-feedback.md)

### Поддержка внешнего микрофона USB-C

> [!IMPORTANT]
> Подключение **USB-микрофона не**будет автоматически устанавливать его в качестве устройства ввода. При подключении набора наушников USB-C пользователи будут наблюдать, что звук наушника автоматически перенаправляется на наушники, но holoLens OS приоритизирует внутренний массив микрофонов над любым другим устройством ввода. **Чтобы использовать микрофон USB-C, выполните следующие действия.**

Пользователи могут выбрать подключенные к USB-C внешние микрофоны с **помощью** панели параметров звука. Микрофоны USB-C можно использовать для звонков, записи и т. д.

Откройте приложение **"Параметры"** и выберите **"Системный**  ->  **звук".**

![Параметры звука](images/usbc-mic-1.jpg)

> [!IMPORTANT]
> Чтобы использовать внешние микрофоны **с**удаленной помощью, пользователям потребуется щелкнуть гиперссылки "Управление звуковых устройствами".
>
> Затем используйте в этом выпадаемом окте для настройки внешнего микрофона значение **по** умолчанию или **по умолчанию для связи.** Выбор значения **по умолчанию** означает, что внешний микрофон будет использоваться везде.
>
> Выбор решения **Communications Default** означает, что внешний микрофон будет использоваться в приложениях удаленной помощи и других приложениях для связи, но микрофонный массив HoloLens может по-прежнему использоваться для выполнения других задач.

![Управление звуковых устройствами](images/usbc-mic-2.png)

#### Как насчет Bluetooth микрофона?

#### Устранение неполадок с микрофонами USB-C

Следует помнить, что некоторые микрофоны USB-C неправильно сообщают о себе как о микрофоне и *динамике.* Это проблема с микрофоном, а не с HoloLens. При подключении одного из этих микрофонов к HoloLens звук может быть потерян. К счастью, существует простое исправление.  

В **параметрах**  ->  **системного**звука явно установите встроенные динамики (аудиофайл аналогового функции) в качестве  ->  **** устройства **по умолчанию.** **** HoloLens должен запомнить этот параметр, даже если микрофон будет удален и повторно подключен позже.

![Устранение неполадок с микрофонами USB-C](images/usbc-mic-4.png)

### Использование новых параметров и приложений Edge в режимах терминала

При добавлении [](hololens-kiosk.md)приложений в киоски ИТ-администратор часто добавляет приложение в терминал, но использует его ИД модели пользователя приложения (AUMID). Так как приложение "Параметры" и приложение Microsoft Edge считаются новыми приложениями и отличаются от того, что старые киоски приложений, которые используют AUMID для этих приложений, должны быть обновлены для использования нового AUMID.

При изменении киоска для добавления новых приложений рекомендуется добавить новый AUMID, а также оставить старый. Это создаст простой переход, когда пользователи обновят ОС и им не потребуется получать новые политики для использования терминала в соответствии с заданной задачей.

| Приложение                    | AUMID                                                  |
|------------------------|--------------------------------------------------------|
| Старое приложение "Параметры"       | HolographicSystemSettings_cw5n1h2txyewy! Приложение            |
| Приложение "Новые параметры"       | BAEAEF15-9BAB-47FC-800B-ACECAD2AE94B_cw5n1h2txyewy! Приложение |
| Старое приложение Microsoft Edge | Microsoft.MicrosoftEdge_8wekyb3d8bbwe!MicrosoftEdge    |
| Новое приложение Microsoft Edge | Microsoft.MicrosoftEdge.Stable_8wekyb3d8bbwe! MSEDGE    |

### Изменения поведения режима терминала для обработки сбоев

В более старых сборках, если на устройстве была конфигурация терминала, которая представляет собой сочетание глобального назначенного доступа и доступа участника группы AAD, если определение членства в группе AAD не удалось, пользователь увидит[в](https://docs.microsoft.com/hololens/hololens-kiosk#kiosk-mode-behavior-changes-for-handling-of-failures)меню "Пуск" ничего не показано.

Начиная с выпуска windows Insider, режим терминала будет откатить к глобальной конфигурации терминала (если она присутствует) в случае сбоев в режиме киоска группы AAD.

## Начало получения сборки для insider

> [!NOTE]
> Если вы не обновились недавно, перезагружайте устройство, чтобы обновить состояние и получить последнюю сборку.
> - Голосовая команда "Перезагружать устройство" работает хорошо. 
> - Вы также можете нажать кнопку перезапуска в программе "Параметры" или "Программа для программы программы для windows".
>
> У нас была ошибка на тыловом сайте, с которой вы могли столкнуться, и это поможет вам вернуться в отслеживание.

On a HoloLens 2 device go to **Settings**  >  **Update & Security**Windows  >  **Insider Program** and select Get **started**. Привяжете учетную запись, которая использовалась для регистрации в качестве пользователя в windows Insider.

В настоящее время в "Каналы" переходит внутренняя пользователи Windows. Круг **Fast** станет каналом **Dev,** круг **Slow** станет каналом **** **Beta,** а круг предварительного просмотра выпуска станет каналом **Release Preview.** Вот как выглядит это сопоставление:

![Описание каналов для insider Windows](images/WindowsInsiderChannels.png)

Дополнительные сведения см. [в канале "Знакомство с Windows Insider Channels" в](https://blogs.windows.com/windowsexperience/2020/06/15/introducing-windows-insider-channels) блогах о Windows.

Затем выберите **"Активная разработка Для Windows",** выберите, хотите ли вы получать сборки канала Dev **или** **Beta Channel,** и просмотрите условия программы.

Выберите **"> перезапустить сейчас",** чтобы завершить работу. После перезагрузки устройства перейдите в **** > "& Безопасность >", чтобы получить последнюю сборку.

### Обновление ошибок 0x80070490 об ошибках
Если при обновлении на канале Dev или Beta 0x80070490 ошибка обновления, попробуйте следующую краткую работу. Он включает перемещение канала для внутренней части, обновление и обратное перемещение канала.

#### Этап 1. Предварительный выпуск
1.  Параметры, обновление & безопасности, программа предварительной подготовки Windows, выберите **канал Release Preview**.
2.  "Параметры", "& безопасность", "Обновление Windows", **"Проверка на обновления".** После обновления пере продолжите работу на этапе 2.

#### Этап 2— канал dev
1. "Параметры", "Обновление & безопасности", "Программа для программы для программы insider **Windows",** выберите канал Dev.
2. "Параметры", "& безопасность", "Обновление Windows", **"Проверка на обновления".**
