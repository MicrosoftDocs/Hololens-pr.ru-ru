# Предварительная версия Microsoft HoloLens

Добро пожаловать в последние сборки Insider Preview для HoloLens!  Вы просто приступите к работе и получите ценную отзыв для следующего основного обновления операционной системы для HoloLens.

Программа предварительной оценки Windows теперь перемещается в каналы. **Быстрый** Звонок становится **каналом разработки**, а **медленный** Звонок становится **каналом бета-версии**, а в качестве **канала**предварительной версии выводится **Звонок.** Ниже показано, как выглядит такое сопоставление:

![Каналы программы предварительной оценки Windows explination](images/WindowsInsiderChannels.png)

Дополнительные сведения: [запись блога Windows](https://blogs.windows.com/windowsexperience/2020/06/15/introducing-windows-insider-channels)

## Начало получения сборок для участников программы предварительной оценки

На устройстве HoloLens 2 перейдите в раздел **Settings**  ->  **Обновление параметров & безопасность**  ->  **программы предварительной оценки Windows** , а затем — **Начало работы**. Свяжите учетную запись, которую вы использовали для регистрации в программе предварительной оценки Windows.

Затем выберите **активную разработку Windows**, укажите, хотите ли вы получать сборки для **канала разработки** или **бета-канала** , и просмотрите условия программы.

Нажмите кнопку **Confirm (подтвердить) > перезапустить** , чтобы завершить работу. После перезагрузки устройства перейдите в раздел **параметры > обновление & безопасность — > проверить наличие обновлений** , чтобы получить последнюю сборку.

## Прекратить получение сборок для участников программы предварительной оценки

Если вы больше не хотите получать сборки участников программы предварительной оценки для Windows holographic, вы можете отказаться от того, когда HoloLens выполняет производственную сборку, или вы можете [восстановить](hololens-recovery.md) устройство с помощью расширенного помощника по восстановлению, чтобы восстановить его до версии Windows holographic без предварительной оценки.

> [!CAUTION]
> Существует известная неполадка, из-за которой пользователи, которые отменяют регистрацию в предварительной версии программы предварительной оценки, после повторной установки новой сборки предварительной версии вручную посталкиваются с синим экраном. После этого они должны вручную восстановить свои устройства. Для получения подробных сведений о том, если вы не повлияет на эту [известную ошибку](https://docs.microsoft.com/hololens/hololens-known-issues?source=docs#blue-screen-is-shown-after-unenrolling-from-insider-preview-builds-on-a-device-reflashed-with-a-insider-build), пожалуйста, ознакомьтесь с дополнительными сведениями.

Чтобы убедиться в том, что HoloLens использует производственную сборку, выполните указанные ниже действия.

1. Перейдите к **параметрам > система > о программе**и найдите номер сборки.
1. [Ознакомьтесь с заметками о выпуске для номеров производственных сборок.](hololens-release-notes.md)

Чтобы отказаться от сборок для участников программы предварительной оценки:

1. На HoloLens, на котором работает производственный процесс, перейдите в раздел **настройки > обновление & безопасность > программы предварительной оценки Windows**и выберите пункт **остановить сборки Insider Preview**.
1. Следуйте указаниям, чтобы отказаться от использования устройства.



## Предоставление отзывов и проблем с отчетами

Для предоставления отзывов и отчетов о проблемах используйте [приложение центра отзывов](hololens-feedback.md) на HoloLens. Используя центр отзывов, вы гарантируете, что все необходимые диагностические сведения будут включены для того, чтобы помочь нашим специалистам быстро отладить и устранить проблему.  Проблемы, связанные с версией HoloLens для китайского и японского языков, должны выводиться одинаковым образом.

> [!NOTE]
> Убедитесь в том, что вы согласны с просьбой получить доступ к папке "документы" в центре отзывов (нажмите кнопку **Да** при появлении соответствующего запроса).

### Поддержка положения автоматического взгляда

В HoloLens 2 положения глаз обеспечивают точное расположение голограмм, удобство просмотра и улучшенное качество изображения. Положения глаз рассчитываются как часть результата отслеживания взгляда. Тем не менее, для этого необходимо, чтобы каждый пользователь просматривает калибровку слежения за глазами, даже если у вас не требуется вводить взгляд.

**Положение с Автовзглядом (AEP)** позволяет этим сценариям без взаимодействия быстро вычислять положения глаз для пользователя.  Автоматическое начало работы с автоматическим глазом начинается с момента, когда пользователь помещает устройство в фоновый режим. Если у пользователя нет более ранней калибровки для отслеживания взгляда, положение с автоглазом начнется с экрана монитора после небольшого времени обработки. Время обработки обычно составляет от 20-60 секунд. Данные пользователя не сохраняются на устройстве и, следовательно, этот процесс повторяется, если пользователь выходит из системы и снова помещает его на устройство или перезапускается из спящего режима.  

Есть несколько изменений поведения системы с функцией положения автоспуска, когда некалиброванный пользователь помещается на устройство. Некалиброванный пользователь обозначает человека, который не прошел процесс калибровки отслеживания взгляда на устройстве ранее.

|     Активное приложение                           |     Текущее поведение                                   |     Поведение в рамках программы предварительной оценки Windows 19041.1339 +                                                      |
|--------------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
|     Приложение, не поддерживающее взгляд, или оболочка holographic    |     Откроется окно калибровки отслеживания взгляда.    |     Запрос не отображается.                                                                                |
|     Приложение для установления взгляда                             |     Откроется окно калибровки отслеживания взгляда.    |     Окно калибровки отслеживания взгляда отображается только в том случае, если приложение получает доступ к потоку взгляда глаз.     |

 Если пользователь переходит из приложения, не поддерживающего взгляд, в приложение, которое обращается к данным взгляда, откроется окно калибровки. Не будет изменено значение "нет на работе". 
 
Для работы, требующей взгляда на глаз или очень точное расположение голограмм, рекомендуется откалибровать пользователей о выполнении калибровки отслеживания взгляда с экрана калибровки слежения за глазами или с помощью приложения "Параметры" в меню "Пуск", а затем выбрав > калибровка **> глаз >** калибровки глаз.

**Известные проблемы**
1.  Мы изучаем проблему, из за которой процесс хоста драйвера средства отслеживания взгляда может аварийно завершить работу при высокой загрузке памяти. Ведущий процесс драйвера отслеживания взглядов должен быть автоматически восстановлен.
