# <a name="insider-preview-for-microsoft-hololens"></a>Предварительная версия Microsoft HoloLens

Добро пожаловать в последние сборки предварительного просмотра для HoloLens! Это просто, чтобы начать [работу](hololens-insider.md#start-receiving-insider-builds) и предоставить ценную обратную связь для нашего следующего крупного обновления операционной системы для HoloLens.

Теперь посетители могут автоматически войти в киоск. Это поведение по умолчанию, но может быть управляемо и отключено.

[**Улучшенная сбой в режиме киоска**](#kiosk-mode-behavior-changes-for-handling-of-failures)

Если членство пользователя AAD в группе AAD не удалось успешно определить, то глобальная конфигурация киоска используется для меню пуск (если присутствует), в противном случае пользователю будет представлено пустое меню пуска. Хотя пустое меню пусков не является конфигурацией, которая может быть непосредственно установлена, эта новая обработка может быть чем-то, чтобы сообщить вашему отделу поддержки о том, используете ли вы киоски, так как это может применяться к конфигурациям или вам может потребоваться внести новые изменения в назначенную конфигурацию доступа.

Несмотря на то, что новый Microsoft Edge в конечном итоге заменит устаревший Microsoft Edge на HoloLens 2, оба браузера в настоящее время доступны для инсайдеров. Пожалуйста, поделитесь отзывами и ошибками с нашей командой с помощью функции **Отправка** отзывов в новом Microsoft Edge или через [центр отзывов](hololens-feedback.md).

#### <a name="what-to-expect-from-the-new-microsoft-edge-on-hololens-2"></a>Чего ожидать от нового Microsoft Edge на HoloLens 2

Поскольку новое microsoft Edge — это родное приложение Win32 с новым слоем адаптеров UWP, позволяющим ему работать на устройствах только для UWP, таких как HoloLens 2, некоторые функции могут быть недоступны сразу. В ближайшие месяцы мы будем поддерживать новые сценарии и функции, поэтому ознакомьтесь с этим пространством для получения самых новых сведений.

**Сценарии и функции, которые должны работать:**
- Первый запуск, вход в профиль и синхронизация
- Веб-сайты должны отрисовки и вести себя так, как ожидалось
- Большинство функций браузера (Избранное, История и т.д.) должны работать так, как ожидалось
- Темный режим
- Установка веб-приложений на устройство
- Установка расширений (пожалуйста, дайте нам знать, если вы используете какие-либо расширения, которые не работают должным образом на HoloLens 2)
- Просмотр и разметка PDF
- Пространственный звук из одного окна браузера
- Автоматическое и ручное обновление браузера
- Сохранение PDF-файла из меню Печать (с помощью параметра "Сохранить в PDF")
- Расширение WebXR и 360 viewer
- Восстановление контента для исправления окна при просмотре нескольких окон, размещенных в среде

**Сценарии и функции, которые не должны работать:**
- Пространственный звук из нескольких окон с одновременными звукопотоками
- "См.
- Вывод на печать

**Известные проблемы браузера:**
- Сброс устройства удалит новый Microsoft Edge
- Предварительный просмотр увеличения в голографической клавиатуре показывает неправильное содержимое
- Прокрутка иногда может заикаться
- Веб-ссылки в приложении Microsoft Store могут не запускать браузер
- Звук может играть из неправильного окна браузера, если вы ранее играли звук из другого окна браузера

#### <a name="microsoft-edge-insider-channels"></a>Каналы инсайдерской службы Microsoft Edge

Команда Microsoft Edge делает три канала предварительного просмотра доступными для сообщества edge Insider: Beta, Dev и Canary. Установка канала предварительного просмотра не отсоединяет выпущенную версию Microsoft Edge на holoLens 2, и вы можете установить несколько одновременно. 

Узнайте больше о сообществе edge [Insider](https://www.microsoftedgeinsider.com) на домашней странице Microsoft Edge Insider. Чтобы узнать больше о различных каналах edge Insider и начать работу, посетите страницу загрузки [edge Insider](https://www.microsoftedgeinsider.com/download).

Существует несколько методов установки каналов microsoft Edge Insider в HoloLens 2:

**Прямая установка на устройстве (в настоящее время доступна только для неуправляемых устройств)**
  1. На странице загрузки Edge [Insider](https://www.microsoftedgeinsider.com/download)2 посетите страницу загрузки HoloLens 2.
  1. Выберите **кнопку Скачать для HoloLens 2 для** канала Edge Insider, который вы хотите установить.
  1. Запустите загруженный файл .msix из очереди загрузки Edge или из папки "Загрузки" устройства (с помощью Проводника файлов).
  1. [Установщик приложения](app-deploy-app-installer.md) запустится.
  1. Выберите **кнопку Установка.**
  1. После успешной установки в списке "Все приложения" меню "Пуск" можно найти бета-версию Microsoft Edge Beta, Dev или Canary. ****

**Установка с помощью компьютера с помощью портала устройств Windows [(требуется](https://docs.microsoft.com/windows/mixed-reality/develop/platform-capabilities-and-apis/using-the-windows-device-portal#setting-up-hololens-to-use-windows-device-portal) включить режим разработчика на HoloLens 2)**
  1. На компьютере посетите страницу [загрузки edge Insider.](https://www.microsoftedgeinsider.com/download)
  1. Выберите **кнопку выпадаемой** стрелки рядом с кнопкой "Скачайте для Windows 10" для канала Edge Insider, который вы хотите установить.
  1. Выберите **HoloLens 2 в** выпадаемом меню.
  1. Сохраните файл .msix в папку "Скачивание" компьютера (или другую папку, которая легко найти).
  1. Используйте [портал устройств Windows на](https://docs.microsoft.com/windows/mixed-reality/develop/platform-capabilities-and-apis/using-the-windows-device-portal#installing-an-app) компьютере для установки загруженного файла .msix на HoloLens 2.
  1. После успешной установки в списке "Все приложения" меню "Пуск" можно найти бета-версию Microsoft Edge Beta, Dev или Canary. ****

> [!NOTE]
> Во время предварительного просмотра Предварительного просмотра Windows для HoloLens 2 версия Microsoft Edge на вашем устройстве может быть выше, чем в некоторых (или всех) каналах предварительного просмотра Microsoft Edge. Это необходимо для обеспечения того, чтобы новые функции и исправления, специально предназначенные для веб-браузера на HoloLens 2, как можно быстрее добирались до наших инсайдеров Windows. Вскоре после публичного выпуска следующего обновления Windows сборки инсайдерского канала Microsoft Edge превзошли версию Microsoft Edge на holoLens 2 и опережают ее.

### <a name="webxr-and-360-viewer"></a>WebXR и 360 Viewer

*Добавлено в Windows Insider build 20289.1000*

Новый Microsoft Edge включает поддержку WebXR, который является новым стандартом для создания иммерсивных веб-опытом (замена Веб-видеорегистратора). Многие иммерсивные веб-опытом были разработаны с учетом виртуальной реальности (они заменяют поле зрения виртуальной средой), но эти впечатления также поддерживаются HoloLens 2. Стандарт WebXR также позволяет использовать веб-среду с расширенной и смешанной реальностью, которая использует физическую среду. Поскольку разработчики проводят больше времени с WebXR, мы ожидаем, что новые возможности погружения в дополненную и смешанную реальность прибудут к клиентам HoloLens 2, чтобы попробовать!

Расширение 360 Viewer построено на WebXR и автоматически устанавливается рядом с новым Microsoft Edge на HoloLens 2. Это веб-расширение позволяет погрузиться в 360-градусные видео. YouTube предлагает самый большой выбор из 360 видео, поэтому мы рекомендуем вам начать с этого.

#### <a name="how-to-use-webxr"></a>Использование WebXR

1. Перейдите на веб-сайт с поддержкой WebXR.
1. Выберите **кнопку Ввод виртуальной реальности** на веб-сайте. Расположение и визуальное представление этой кнопки могут отличаться на веб-сайте, но могут выглядеть так же, как:

    ![Введите пример кнопки VR](images/75px-enter-vr.png)

1. При первом запуске webXR-приложения в определенном домене браузер запросит согласие на ввод иммерсивного представления, выберите **Разрешить**.
1. Чтобы [управлять опытом, используйте жесты HoloLens 2.](hololens2-basic-usage.md#the-hand-tracking-frame)
1. Если у этого опыта нет кнопки **Exit,** используйте жест [Начните](hololens2-basic-usage.md#start-gesture) возвращаться домой.

**Рекомендуемые образцы WebXR**
- 360 Viewer (см. следующий раздел)
- [Динозавры XR](https://www.xrdinosaurs.com/)
- [Экспресс Бариста](https://constructarca.de/game/barista-express/)
- [Краска WebXR](https://threejs.org/examples/webxr_vr_paint.html)

#### <a name="how-to-use-360-viewer"></a>Использование 360 Viewer

1. Перейдите к 360-градусной видеозаписи на YouTube.
1. В кадре видео выберите кнопку гарнитуры смешанной реальности:

    ![Кнопка активации 360 Viewer](images/enter-360-viewer.jpg)

1. При первом запуске 360 Viewer в определенном домене браузер запросит согласие на ввод иммерсивного представления. Выберите **Разрешить**.
1. [Касание](hololens2-basic-usage.md#select-using-air-tap) воздуха для воспроизведения элементов управления. Используйте [лучи рук](hololens2-basic-usage.md#select-using-air-tap) и касание воздуха для воспроизведения/паузы, пропускания вперед/назад, включите/выключайте подписи или остановите впечатление (которое выходит из иммерсивного представления). Элементы управления воспроизведением исчезнут через несколько секунд после бездействия.

#### <a name="top-webxr-and-360-viewer-known-issues"></a>Известные проблемы с top webXR и 360 viewer
- В опытом WebXR при наклоне головы или движении вокруг среды голограммы могут перемещаться или наклоняться.
- В зависимости от сложности работы с WebXR рамка может упасть или заикаться.
- Артикуляционные суставы руки еще не доступны в WebXR.
- При выходе из webXR или 360 viewer может потребоваться 30 секунд или более, чтобы голограммы в доме смешанной реальности появлялись повторно.
- 360 видео с веб-сайтов, помимо YouTube, могут работать не так, как ожидалось.
- Если 360 видео не введите иммерсивное представление (или кнопка гарнитуры смешанной реальности не появится), попробуйте освежить страницу.
- Подписи еще не видны в 360 Viewer на HoloLens 2.
- Приодержание видео в 360 Viewer останавливает отрисовку видео (но правильный выбор кнопки воспроизведения возобновляет воспроизведение).
- Кнопка "Следующее видео" в 360 Viewer в настоящее время не работает.
- Вы можете играть 2D-видео в режиме иммерсивного "театра", но частота кадров будет меньше 30 fps.

### <a name="display-color-calibration"></a>Отображение калибровки цветов

*Добавлено в Windows Insider build 20293.1000*

С помощью этого нового параметра можно выбрать альтернативный цветной профиль для отображения HoloLens 2. Это может помочь цветам выглядеть более точными, особенно при более низких уровнях яркости отображения. Отобразить калибровку цветов можно в приложении Параметры, на странице > калибровка.

> [!NOTE]
> Так как этот параметр сохраняет новый профиль цвета для прошивки дисплея, он является параметром для каждого устройства (и не является уникальным для каждой учетной записи пользователя).

#### <a name="how-to-use-display-color-calibration"></a>Использование калибровки цвета отображения

1. Запустите **приложение Параметры** и перейдите в **system > калибровку.**
1. В **соответствии с калибровка цвета**отображения выберите кнопку **"Выполнить калибровку цвета отображения".**
1. Опыт калибровки цвета отображения запустится и поможет вам убедиться, что ваш козырек находится в правильном положении.
1. После того как вы переходите через диалоговое окно инструкции, ваш дисплей автоматически будет затемнеть до 30% яркости.
    > [!TIP]
    > Если у вас возникли проблемы с темной сценой в среде, можно вручную настроить уровень яркости HoloLens 2 с помощью кнопок яркости в левой части устройства.
1. Выберите кнопки 1-6, чтобы мгновенно опробовать каждый профиль цвета, и найти тот, который лучше всего смотрится в глаза (обычно это означает, что профиль, который помогает сцене выглядеть наиболее нейтрально, с серым узором и тонами кожи, как ожидалось.)

    ![Отображение сцены калибровки цветов](images/color-cal-ui.png)
    
1. Если вы довольны выбранным профилем, выберите кнопку Сохранить & **exit**
1. Если вы предпочитаете не вносить изменения, выберите кнопку **Отмена** & exit и изменения будут отменены.

> [!TIP]
> Вот несколько полезных советов, которые необходимо иметь в виду при использовании параметра калибровки цвета отображения:
> - Вы можете повторно запустить калибровку цвета отображения из параметров, когда захотите
> - Если кто-либо на устройстве ранее использовал параметр для изменения профилей цветов, дата и время последнего изменения будут отражены на странице Параметры
> - При повторном запуске калибровки цвета отображения будет выделен профиль цвета, который был сохранен ранее, и профиль 0 не будет отображаться (так как профиль 0 представляет исходный профиль цвета)
> - Если вы хотите вернуться к исходному цвету отображения, вы можете сделать это со страницы Параметры (см. как [сбросить цветной профиль)](#how-to-reset-color-profile)

#### <a name="how-to-reset-color-profile"></a>Сброс цветового профиля

Если вы недовольны пользовательским профилем цвета, сохраненным в holoLens 2, вы можете восстановить исходный профиль цвета устройства:
1. Запустите **приложение Параметры** и перейдите в **system > калибровку.**
1. В **соответствии с калибровка цвета**отображения выберите кнопку Сброс по **умолчанию.**
1. Когда откроется диалоговое окно, выберите **перезапуск,** если вы готовы перезапустить HoloLens 2 и применить изменения.

#### <a name="top-display-color-calibration-known-issues"></a>Известные проблемы калибровки цветов верхнего отображения

- На странице Параметры строка состояния, которая сообщает, когда был изменен последний профиль цвета, будет устарела до тех пор, пока вы не перезагрузите эту страницу Параметров 
    - Обходное решение. Выберите другую страницу Параметры, а затем повторно выберите страницу калибровки.
- Если ваш HoloLens 2 заснул во время работы калибровки цвета отображения, он затем возобновится в дом смешанной реальности, и уровень яркости дисплея будет по-прежнему притуплен.
- Вам может потребоваться несколько раз нажать кнопки яркости на левой стороне устройства, прежде чем они будут работать как ожидалось.
- Локализация не завершена для всех рынков

### <a name="swipe-to-type"></a>Проведите пальцем по типу

Некоторые клиенты быстрее "введите" на виртуальных клавиатурах, проводя фигуру слова, которое они собираются ввести, и мы просматриваем эту функцию для голографической клавиатуры. Чтобы протестировать с помощью ffu с подписанным рейсом, сначала необходимо разблокировать устройство перед мигалком подписанного на рейсе ffu.
1. На ПК:

    1. Скачайте ffu на компьютер с [https://aka.ms/hololenspreviewdownload](https://aka.ms/hololenspreviewdownload) .
    
    1. Установка ARC (Advanced Recovery Companion) из Microsoft Store: [https://www.microsoft.com/store/productId/9P74Z35SFRS8](https://www.microsoft.com/store/productId/9P74Z35SFRS8) .
    
1. На HoloLens — разблокировка полета: обновление **** открытых параметров & программы инсайдерской защиты Windows, затем зарегистриройся, перезагрузка  >  ****  >  **** устройства.

1. Flash FFU . Теперь вы можете вспыхить полет, подписанный FFU с помощью ARC.

## <a name="provide-feedback-and-report-issues"></a>Предоставление отзывов и отчетов о проблемах

Используйте приложение [Концентратор отзывов](hololens-feedback.md) в holoLens для предоставления отзывов и отчетов о проблемах. Использование Центра отзывов гарантирует, что вся необходимая диагностическая информация включена, чтобы помочь нашим инженерам быстро отладить и устранить проблему.  Проблемы с китайской и японской версией HoloLens должны быть сообщаться таким же образом.

> [!NOTE]
> Убедитесь, что вы принимаете запрос на запрос, хотите ли вы получить доступ к папке "Документы" (выберите **Да** по запросу).

## <a name="note-for-developers"></a>Примечание для разработчиков

Вы можете попробовать разработку приложений с помощью сборки HoloLens с помощью insider.  Ознакомьтесь с [документацией разработчика HoloLens для](https://developer.microsoft.com/windows/mixed-reality/development) начала работы. Эти же инструкции работают с инсайдерской сборкой HoloLens.  Вы можете использовать те же сборки Unity и Visual Studio, которые уже используются для разработки HoloLens.

## <a name="stop-receiving-insider-builds"></a>Остановка получения сборки insider

Если вы больше не хотите получать инсайдерские сборки Windows Holographic, вы можете отказаться от использования [](hololens-recovery.md) сборки holoLens или восстановить устройство с помощью advanced Recovery Companion для восстановления устройства в версии Windows Holographic, которая не является инсайдерской.

> [!CAUTION]
> Существует известная проблема, из-за которой пользователи, которые после ручного переустановки новой сборки предварительного просмотра отчислений из предварительного просмотра будут создаваться с синим экраном. После этого они должны вручную восстановить свое устройство. Дополнительные сведения о том, повлияет ли на вас влияние или нет, можно просмотреть дополнительные сведения по этой [известной проблеме.](https://docs.microsoft.com/hololens/hololens-known-issues?source=docs#blue-screen-is-shown-after-unenrolling-from-insider-preview-builds-on-a-device-reflashed-with-a-insider-build)

Чтобы убедиться, что holoLens запущена производственная сборка:

1. Перейдите **к параметрам > system > About**и найдите номер сборки.

1. [См. заметки о выпуске для номеров сборки производства.](hololens-release-notes.md)

Чтобы отказаться от сборки insider:

1. В holoLens, запускаемом производственной сборкой, перейдите в параметры > **Update & Security > Windows Insider Program**и выберите **сборки Stop Insider**.

1. Следуйте инструкциям, чтобы отказаться от устройства.
