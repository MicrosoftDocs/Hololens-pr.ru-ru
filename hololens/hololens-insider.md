### Знакомство с новым Microsoft Edge

![Анимация устаревшего логотипа Microsoft Edge до нового логотипа Microsoft Edge](images/new-edge.gif)

Новый microsoft Edge принимает проект с открытым исходным кодом [Chromium,](https://blogs.windows.com/windowsexperience/2018/12/06/microsoft-edge-making-the-web-better-through-more-open-source-collaboration/) чтобы улучшить совместимость для клиентов и менее фрагментацию веб-сайта для веб-разработчиков. 

С помощью этой предварительной версии новый Microsoft Edge доступен пользователям HoloLens 2 в первый раз! Хотя новый Microsoft Edge в конечном итоге заменит устаревший Microsoft Edge на HoloLens 2, оба браузера в настоящее время доступны для пользователей программы программы. Поделитесь своим мнением и ошибками с нашей командой с помощью функции **отправки отзывов** в новом Microsoft Edge или в [Центре отзывов.](hololens-feedback.md)

![Новый снимок экрана Microsoft Edge](images/new-edge-ui.png)

#### Запуск нового Microsoft Edge

Для программы insiders доступны две версии Microsoft Edge: новый значок Microsoft Edge (представленный синим и зеленым значком значка ![ ](images/new_edge_logo.png) "e"). Новый Microsoft Edge закреплен в меню "Пуск" и автоматически запускается при активации веб-ссылки. Если вы хотите вернуться к использованию устаревшей службы Microsoft Edge в качестве веб-браузера по умолчанию, см. инструкции ниже по сбросу приложений [по умолчанию.](#default-app-picker)

> [!NOTE]
> При первом запуске нового Microsoft Edge на HoloLens 2 параметры и данные будут импортироваться из устаревшей службы Microsoft Edge. Если вы продолжите использовать устаревший Microsoft Edge после запуска нового Microsoft Edge, эти новые данные не будут синхронизироваться с новым Microsoft Edge из устаревшей службы Microsoft Edge.

#### Настройка параметров политики для нового Microsoft Edge

Новый Microsoft Edge предлагает ИТ-специалистов гораздо более широкий набор политик браузера на HoloLens 2, чем ранее был доступен в Microsoft Edge прежних версиях. 

Вот некоторые полезные ресурсы для получения дополнительных знаний об управлении настройками политики для нового Microsoft Edge:
- [Настройка параметров политики Microsoft Edge с помощью Microsoft Intune](https://docs.microsoft.com/deployedge/configure-edge-with-intune)
- [Сопоставление политик устаревшей версии Microsoft Edge и Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge)
- [Сопоставление политик Google Chrome и Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-chrome-to-newedge)
- Полная [документация по Microsoft Edge Enterprise](https://docs.microsoft.com/deployedge/)

> [!IMPORTANT]
> Из-за большого объема политик браузера, поддерживаемых новым Microsoft Edge, наша команда не может гарантировать, что каждая новая политика работает на HoloLens 2. Однако мы протестировали и проверили, что новый эквивалент Microsoft Edge для каждой устаревшей политики Microsoft Edge, ранее поддерживаемой на HoloLens 2, работает ожидаемым образом. См. [сопоставление](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge) политик Microsoft Edge Legacy и Microsoft Edge, чтобы найти новый эквивалент Microsoft Edge для каждой устаревшей политики браузера Microsoft Edge, которая использовался с HoloLens 2.
>
> Существует по крайней мере две новые политики Microsoft Edge, которые не *будут* работать с HoloLens 2:
> - EnterpriseModeSiteList
> - EnterpriseSiteListServiceURL

#### Чего ожидать от нового Microsoft Edge на HoloLens 2

Так как новый Microsoft Edge — это приложение Win32 с новым слоем адаптеров UWP, позволяющим работать только на устройствах UWP, таких как HoloLens 2, некоторые функции могут быть недоступны сразу. В ближайшие месяцы мы будем поддерживать новые сценарии и функции, поэтому ознакомьтесь с этой информацией в этом пространстве.

**Сценарии и функции, которые должны работать:**
- Первый запуск, вход в профиль и синхронизация
- Веб-сайты должны отрисовки и вести себя ожидаемым образом
- Большинство функций браузера (избранное, история и т. д.) должны работать ожидаемым образом.
- Темный режим
- Установка веб-приложений на устройство
- Установка расширений (сообщите нам, если вы используете какие-либо расширения, которые не работают должным образом на HoloLens 2)
- Просмотр и разметка PDF-файла
- Пространственный звук из одного окна браузера
- Автоматическое и ручное обновление браузера
- Сохранение PDF-файла из меню "Печать" (с помощью параметра "Сохранить в ФОРМАТ PDF")

**В ближайшее время будут представлены сценарии и функции:**
- WebXR и расширение viewer 360
- Восстановление содержимого для корректного окна при просмотре по нескольким окнам, размещенным в вашей среде
- Пространственный звук для нескольких окон с одновременными аудиопотоками
- Присоединение к вызову Microsoft Teams через браузер с помощью видео, смешанного захвата реальности или совместного использования экрана (присоединение звонков со звуком работает хорошо)
- "See it, say it" (См.
- Вывод на печать

**Известные проблемы браузера:**
- Сброс устройства удалит новый Microsoft Edge
- Предварительный просмотр лубки на голографической клавиатуре показывает неправильное содержимое

### Приложение "Новые параметры"

В этом выпуске мы представляем новую версию приложения "Параметры". Новое приложение "Параметры" включает новые функции и расширенные параметры holoLens 2 в следующих областях: звуковые устройства ввода и вывода, отдельный том приложения, питание и спящий режим, адаптер Ethernet, простота доступа, режим "самолет" и приложения по умолчанию.

> [!NOTE]
> Так как новое приложение "Параметры" отличается от устаревшего приложения "Параметры", все окна параметров, ранее размещенные в среде, будут удалены после обновления.

![Домашняя страницы приложения "Новые параметры"](images/new-settings-app.png)

**Новые функции и параметры**
- Поиск параметров: поиск параметров на домашней странице "Параметры" с помощью ключевых слов или имени параметра
- Звук:
  - Звуковые устройства ввода и вывода: независимо выберите звуковые устройства ввода и вывода (например, прослушивание звука через наушники Bluetooth наушников или использование микрофона USB-C для ввода звука). Примечание. Bluetooth микрофоны не поддерживаются HoloLens 2.
  - Объем приложения: независимо отрегулировать объем каждого приложения
- Экономия заряда: вручную включите режим экономии заряда или установите пороговое значение заряда батареи, при котором режим экономии заряда включается автоматически
- Питание &: выберите, когда устройство должно перейти в спящий режим после периода бездействия
- USB: вы можете отключить USB-подключения по умолчанию
- Сетевые & Интернет:
  - Адаптеры Ethernet USB-C теперь отображаются в сетевом & Интернете
  - Теперь доступны параметры адаптеров Ethernet USB-C, включая ЕГО IP-адрес
  - Теперь можно включить режим "в самолете" на HoloLens 2
- Приложения: вы можете сбросить приложения по умолчанию, используемые для типов файлов и ссылок. Дополнительные [сведения см.](#default-app-picker) в стандартной области "Выбор приложений".
- Простота доступа: изменение размера текста и некоторых визуальных эффектов

**Известные проблемы**
- Ранее размещенные окна параметров будут удалены (см. примечание выше)
- При посещении страницы "Уведомления" может возникнуть сбой приложения "Параметры" (исследование)
- Страница Ethernet в настоящее время не появится (скоро она будет исправлена)
- Использование аккумулятора для нового Microsoft Edge может быть не точным, так как это так, так как настольное приложение Win32 поддерживается на уровне адаптеров UWP (скоро исправление не ожидается)

### Выбор приложений по умолчанию

Когда вы активируете гиперссылки или открываете тип файла с более чем одним установленным приложением, которое поддерживает ее, откроется новое окно с запросом на выбор установленного приложения, которое должно обрабатывать файл или тип ссылки. В этом окне можно также выбрать, чтобы выбранное приложение обрабатывал файл или тип ссылки "Once" или "Always". 

![Окно "Выбор приложений"](images/default-app-picker.png)

Если вы выбрали "Всегда", но позже захотите изменить, какое приложение обрабатывает определенный тип файла или ссылки, вы можете сбросить сохраненные значения по умолчанию в параметрах **> Apps.** Прокрутите страницу вниз **** и выберите кнопку "Очистить" в области "Приложения по умолчанию для типов файлов" и/или "Приложения по умолчанию для типов ссылок". В отличие от аналогичного параметра на настольных компьютерах, нельзя сбросить значения по умолчанию для отдельных типов файлов.

### Office Web App

Веб-приложение Office добавлено в список "Все приложения" в меню "Пуск". Это веб-приложение также можно закрепить в start или uninstalled. Так как это веб-приложение, его функциональные возможности в точности совпадают с возможностями, которые вы бы могли испытать при https://www.office.com посещении. Функции Office Web App доступны, только если у HoloLens 2 активно подключение к Интернету.

### Прокрутка для введите

Некоторые клиенты быстрее "введите" виртуальные клавиатуры, протирая форму слова, которое они намерены ввести, и мы просматриваем эту функцию для голографической клавиатуры. Вы можете провести пальцем по одному слову, передав кончик пальца по плоскости голографической клавиатуры, прокрутирав форму слова, а затем выведя кончик пальца из плоскости клавиатуры. Вы можете прокрутки слов, не нажимая пробел, удалив палец от клавиатуры между словами. Вы будете знать, что эта функция работает, если вы видите след прокрутки после движения пальца на клавиатуре.

Обратите внимание, что эта функция может быть сложной для использования и хозяина из-за природы голографической клавиатуры, где вы не будете противяться пальцу (в отличие от дисплея мобильного телефона). Мы оцениваем эту функцию для общедоступных выпусков, поэтому ваш отзыв важен; Независимо от того, полезна ли функция или у вас есть обратная связь по отзывам, сообщите нам об этом через [Центр отзывов.](hololens-feedback.md)





## Начало получения сборки для insider

> [!NOTE]
> Если вы еще не обновились недавно, перезагружайте устройство, чтобы обновить состояние и получить последнюю сборку.
> - Голосовая команда "Перезагрузка устройства" работает хорошо. 
> - Вы также можете нажать кнопку перезапуска в программе "Параметры" или "Программа для программы программы для windows".
>
> У нас была ошибка на тыловом сайте, с которой вы могли столкнуться, и это поможет вам вернуться в отслеживание.

On a HoloLens 2 device go to **Settings**  >  **Update & Security**Windows  >  **Insider Program** and select Get **started**. Привяжете учетную запись, которая использовалась для регистрации в качестве пользователя в windows Insider.

В настоящее время в "Каналы" переходит внутренняя система Windows. Круг **Fast** станет каналом **Dev,** круг **Slow** станет каналом **** **Beta,** а круг предварительного просмотра выпуска станет каналом **Release Preview.** Вот как выглядит это сопоставление:

![Описание каналов для insider Windows](images/WindowsInsiderChannels.png)

Дополнительные сведения см. [в канале "Знакомство с Windows Insider Channels" в](https://blogs.windows.com/windowsexperience/2020/06/15/introducing-windows-insider-channels) блогах о Windows.

Затем выберите **"Активная разработка Для Windows",** выберите, хотите ли вы получать сборки канала Dev **или** **Beta Channel,** и просмотрите условия программы.

Выберите **"> перезапустить сейчас",** чтобы завершить работу. После перезагрузки устройства перейдите в **** > "& Безопасность" > "Проверка обновлений для получения последней сборки".

## Маршруты загрузки и флэш-накопителя FFU
Чтобы протестировать ffu со подписью тестового устройства, сначала необходимо разблокировать устройство, прежде чем мигать ffu с подписью тестового устройства.
1. На компьютере:

    1. Скачайте ffu на компьютер с [https://aka.ms/hololenspreviewdownload](https://aka.ms/hololenspreviewdownload) .
    
    1. Установите ARC (advanced Recovery Companion) из Microsoft Store: [https://www.microsoft.com/store/productId/9P74Z35SFRS8](https://www.microsoft.com/store/productId/9P74Z35SFRS8)
    
1. On HoloLens - Flight Unlock: Open **Settings**  >  **Update & Security**Windows  >  **Insider Program** then sign up, reboot device.

1. Flash FFU — теперь вы можете flash the flight signed FFU using ARC.

> [!CAUTION]
> Существует известная проблема, из-за которой пользователи, которые отостановят регистрацию в сборках Insider Preview после ручной переустановки новой сборки предварительного просмотра, будут испытывать синий экран. После этого им необходимо вручную восстановить устройство. Дополнительные сведения о том, как это повлияет на вас, можно просмотреть по этой [известной проблеме.](https://docs.microsoft.com/hololens/hololens-known-issues?source=docs#blue-screen-is-shown-after-unenrolling-from-insider-preview-builds-on-a-device-reflashed-with-a-insider-build)

Чтобы убедиться, что на вашем HoloLens запущена производственная сборка:

1. Перейдите **в > system > About**и найдите номер сборки.

1. [См. заметки о выпуске производственных номеров сборки.](hololens-release-notes.md)

Чтобы отказаться от сборки для insider:

1. On a HoloLens running a production build, go to **Settings > Update & Security > Windows Insider Program**, and select **Stop Insider builds**.

1. Следуйте инструкциям, чтобы отказаться от устройства.
