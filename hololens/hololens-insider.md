#### Запуск нового Microsoft Edge

Для программы insiders доступны две версии Microsoft Edge: новый значок Microsoft Edge (представленный синим и зеленым значком значка ).) и устаревший Microsoft Edge (представленный белым значком ![ ](images/new_edge_logo.png) "e"). Новый Microsoft Edge закреплен в меню "Пуск" и автоматически запускается при активации веб-ссылки. Если вы хотите вернуться к использованию Устаревшего Браузера Microsoft Edge в качестве веб-браузера по умолчанию, см. инструкции ниже для сброса приложений [по умолчанию.](#default-app-picker)

> [!NOTE]
> При первом запуске нового Microsoft Edge на HoloLens 2 параметры и данные будут импортироваться из устаревшей службы Microsoft Edge. Если вы продолжите использовать устаревший Microsoft Edge после запуска нового Microsoft Edge, эти новые данные не будут синхронизироваться с новым Microsoft Edge из устаревшей службы Microsoft Edge.

Новый Microsoft Edge предлагает ИТ-администраторам гораздо более широкий набор политик браузера на HoloLens 2, чем ранее был доступен в Microsoft Edge прежних версиях.

Вот некоторые полезные ресурсы для получения дополнительных знаний об управлении настройками политики для нового Microsoft Edge:

- [Сопоставление политик Google Chrome и Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-chrome-to-newedge)

> Из-за большого объема политик браузера, поддерживаемых новым Microsoft Edge, наша команда не может гарантировать, что каждая новая политика работает на HoloLens 2. Однако мы протестировали и проверили, что новый эквивалент Microsoft Edge для каждой устаревшей политики Microsoft Edge, ранее поддерживаемой на HoloLens 2, работает ожидаемым образом. #### Чего ожидать от нового Microsoft Edge на HoloLens 2

Так как новый Microsoft Edge — это приложение Win32 с новым слоем адаптеров UWP, позволяющим работать только на устройствах UWP, таких как HoloLens 2, некоторые функции могут быть недоступны сразу. В ближайшие месяцы мы будем поддерживать новые сценарии и функции, поэтому ознакомьтесь с этой информацией в этом пространстве.

**Сценарии и функции, которые должны работать:**
- Первый запуск, вход в профиль и синхронизация
- Веб-сайты должны отрисовки и вести себя ожидаемым образом
- Большинство функций браузера (избранное, история и т. д.) должны работать ожидаемым образом.
- Темный режим
- Установка веб-приложений на устройство
- Установка расширений (сообщите нам, если вы используете какие-либо расширения, которые не работают должным образом на HoloLens 2)
- Просмотр и разметка PDF-файла
- Пространственный звук из одного окна браузера
- Автоматическое и ручное обновление браузера
- Сохранение PDF-файла из меню "Печать" (с помощью параметра "Сохранить в ФОРМАТ PDF")

**В ближайшее время будут представлены сценарии и функции:**
- WebXR и расширение viewer 360
- Восстановление содержимого для корректного окна при просмотре по нескольким окнам, размещенным в вашей среде
- Присоединение к вызову Microsoft Teams через браузер с помощью видео, смешанного захвата реальности или совместного использования экрана (присоединение звонков со звуком работает хорошо)

**Сценарии и функции не должны работать:**
- Пространственный звук из нескольких окон с одновременными аудиопотоками
- "See it, say it" (См.
- Вывод на печать

**Известные проблемы браузера:**
- Сброс устройства удалит новый Microsoft Edge
- Предварительный просмотр лубки на голографической клавиатуре показывает неправильное содержимое

### Приложение "Новые параметры"

В этом выпуске мы представляем новую версию приложения "Параметры". Новое приложение "Параметры" включает новые функции и расширенные параметры HoloLens 2 в следующих областях: "Звук", "Power & спящий режим", "Сетевой & Интернет", "Приложения", "Учетные записи", "Простота доступа" и другие.

> [!NOTE]
> Так как новое приложение "Параметры" отличается от устаревшего приложения "Параметры", все окна параметров, ранее размещенные в среде, будут удалены после обновления.

![Домашняя страницы приложения "Новые параметры"](images/new-settings-app.png)

**Новые функции и параметры**
- Поиск параметров: поиск параметров на домашней странице "Параметры" с использованием ключевых слов или имени параметра
- Системный > звук:
  - Звуковые устройства ввода и вывода: независимо выберите звуковые устройства ввода и вывода (например, прослушивание звука через наушники Bluetooth наушников или использование микрофона USB-C для ввода звука). Примечание. Bluetooth микрофоны не поддерживаются HoloLens 2.
  - Громкость приложения: независимо отрегулировать объем каждого приложения
- Системные > power & спящий режим: выберите, когда устройство должно перейти в спящий режим после периода бездействия
- Системная > батареи: вручную включите режим экономии заряда или установите пороговое значение заряда батареи, при котором режим экономии заряда включается автоматически
- Устройства > USB: по умолчанию можно отключить USB-подключения
- Сетевые & Интернет:
  - Адаптеры Ethernet USB-C теперь отображаются в сетевом & Интернете
  - Теперь доступны параметры адаптеров Ethernet USB-C, включая ЕГО IP-адрес
  - Теперь можно включить режим "в самолете" на HoloLens 2
- Приложения: вы можете сбросить приложения по умолчанию, используемые для типов файлов и ссылок. Дополнительные [сведения см.](#default-app-picker) в стандартной области "Выбор приложений".
- Учетные записи > других пользователей: владельцы устройств могут добавлять пользователей, обновлять стандартных пользователей до владельцев устройств, понизить уровень владельцев устройств до обычных пользователей и удалить пользователей.
- Простота доступа: изменение размера текста и некоторых визуальных эффектов

**Известные проблемы**
- Ранее размещенные окна параметров будут удалены (см. примечание выше)
- При посещении страницы "Уведомления" может возникнуть сбой приложения "Параметры" (исследование)
- Страница Ethernet в настоящее время не появится (скоро она будет исправлена)
- Использование батареи для нового Microsoft Edge может быть не точным из-за его природы в качестве настольного приложения Win32, поддерживаемого слоем адаптеров UWP (скоро исправление не ожидается)

### Выбор приложения по умолчанию

При активации гиперссылки или открытом типе файла с более чем одним установленным приложением, которое поддерживает ее, откроется новое окно с запросом на выбор установленного приложения, которое должно обрабатывать файл или тип ссылки. В этом окне можно также выбрать, чтобы выбранное приложение обрабатывал файл или тип ссылки "Once" или "Always".

![Окно "Выбор приложений"](images/default-app-picker.png)

Если вы выберете "Всегда", но позже захотите изменить, какое приложение обрабатывает определенный тип файла или ссылки, вы можете сбросить сохраненные значения по умолчанию в параметрах **> Apps.** Прокрутите страницу вниз **** и выберите кнопку "Очистить" в области "Приложения по умолчанию для типов файлов" и/или "Приложения по умолчанию для типов ссылок". В отличие от аналогичного параметра на настольных компьютерах, нельзя сбросить значения по умолчанию для отдельных типов файлов.

### Прокрутка для введите

Некоторые клиенты быстрее "введите" виртуальные клавиатуры, протирая форму слова, которое они намерены ввести, и мы просматриваем эту функцию для голографической клавиатуры. Вы можете провести пальцем по одному слову, передав кончик пальца по плоскости голографической клавиатуры, прокрутирав форму слова, а затем выведя кончик пальца из плоскости клавиатуры. Вы можете провести пальцем по словам, не нажимая клавишу пробела, удалив палец от клавиатуры между словами. Вы будете знать, что эта функция работает, если вы видите след прокрутки после движения пальца на клавиатуре.

Обратите внимание, что эта функция может быть сложной для использования и хозяина из-за природы голографической клавиатуры, где вы не будете противяться пальцу (в отличие от дисплея мобильного телефона). Мы оцениваем эту функцию для общедоступных выпусков, поэтому ваш отзыв важен; Независимо от того, полезна ли функция или у вас есть обратная связь по отзывам, сообщите нам об этом через [Центр отзывов.](hololens-feedback.md)

### Поддержка внешнего микрофона USB-C

> [!IMPORTANT]
> Подключение **USB-микрофона не**будет автоматически устанавливать его в качестве устройства ввода. При подключении набора наушников USB-C пользователи будут наблюдать, что звук наушника автоматически перенаправляется на наушники, но holoLens OS приоритизирует внутренний массив микрофонов над любым другим устройством ввода. **Чтобы использовать микрофон USB-C, выполните следующие действия.**

Пользователи могут выбрать подключенные к USB-C внешние микрофоны с **помощью** панели параметров звука. Микрофоны USB-C можно использовать для звонков, записи и т. д.

Откройте приложение **"Параметры"** и выберите **"Системный**  ->  **звук".**

![Параметры звука](images/usbc-mic-1.jpg)

> [!IMPORTANT]
> Чтобы использовать внешние микрофоны **с**удаленной помощью, пользователям потребуется щелкнуть гиперссылки "Управление звуковых устройствами".
>
> Затем используйте в этом выпадании, **** чтобы установить внешний микрофон по умолчанию или **по умолчанию для связи.** Выбор значения **по умолчанию** означает, что внешний микрофон будет использоваться везде.
>
> Выбор решения **Communications Default** означает, что внешний микрофон будет использоваться в приложениях удаленной помощи и других приложениях для связи, но микрофонный массив HoloLens может по-прежнему использоваться для выполнения других задач.

#### Как насчет Bluetooth микрофона?

К Bluetooth микрофоны по-прежнему не поддерживаются на HoloLens 2.

#### Устранение неполадок с микрофонами USB-C

Следует помнить, что некоторые микрофоны USB-C неправильно сообщают о себе как о микрофоне и *динамике.* Это проблема с микрофоном, а не с HoloLens. При подключении одного из этих микрофонов к HoloLens звук может быть потерян. К счастью, существует простое исправление.  

В **параметрах**  ->  **системного**звука явно установите встроенные динамики (аудиофайл аналогового функции) в качестве  ->  **** устройства **по умолчанию.** **** HoloLens должен запомнить этот параметр, даже если микрофон будет удален и повторно подключен позже.

### Использование новых параметров и приложений Edge в режимах терминала

При добавлении [](hololens-kiosk.md)приложений в киоски ИТ-администратор часто добавляет приложение в терминал, но использует его ИД модели пользователя приложения (AUMID). Так как приложение "Параметры" и приложение Microsoft Edge считаются новыми приложениями и отличаются от того, что старые киоски приложений, которые используют AUMID для этих приложений, должны быть обновлены для использования нового AUMID.

При изменении киоска для добавления новых приложений рекомендуется добавить новый AUMID, а также оставить старый. Это создаст простой переход, когда пользователи обновят ОС и им не потребуется получать новые политики для использования терминала в соответствии с заданной задачей.

| Приложение                    | AUMID                                                  |
|------------------------|--------------------------------------------------------|
| Старое приложение "Параметры"       | HolographicSystemSettings_cw5n1h2txyewy! Приложение            |
| Приложение "Новые параметры"       | BAEAEF15-9BAB-47FC-800B-ACECAD2AE94B_cw5n1h2txyewy! Приложение |
| Старое приложение Microsoft Edge | Microsoft.MicrosoftEdge_8wekyb3d8bbwe!MicrosoftEdge    |
| Новое приложение Microsoft Edge | Microsoft.MicrosoftEdge.Stable_8wekyb3d8bbwe! MSEDGE    |

### Изменения поведения режима терминала для обработки сбоев

В более старых сборках, если на устройстве была конфигурация терминала, которая представляет собой сочетание глобального назначенного доступа и доступа участника группы AAD, если определение членства в группе AAD не удалось, пользователь увидит[в](https://docs.microsoft.com/hololens/hololens-kiosk#kiosk-mode-behavior-changes-for-handling-of-failures)меню "Пуск" ничего не показано.

Начиная с выпуска windows Insider, режим терминала будет откатить к глобальной конфигурации терминала (если она присутствует) в случае сбоев в режиме киоска группы AAD.

### Настройка отката диагностики с помощью приложения "Параметры"

Теперь в приложении "Параметры" пользователь может настроить поведение средств диагностики [отката.](hololens-diagnostic-logs.md) В приложении "Параметры" перейдите на **страницу "Устранение**  ->  **неполадок** конфиденциальности", чтобы настроить этот параметр.

> [!NOTE]
> Если для устройства настроена политика MDM, пользователь не сможет переопределять это поведение.  






## Начало получения сборки для insider

> [!NOTE]
> Если вы не обновились недавно, перезагружайте устройство, чтобы обновить состояние и получить последнюю сборку.
> - Голосовая команда "Перезагружать устройство" работает хорошо. 
> - Вы также можете нажать кнопку перезапуска в программе "Параметры" или "Программа для программы программы для windows".
>
> У нас была ошибка на тыловом сайте, с которой вы могли столкнуться, и это поможет вам вернуться в отслеживание.

On a HoloLens 2 device go to **Settings**  >  **Update & Security**Windows  >  **Insider Program** and select Get **started**. Привяжете учетную запись, которая использовалась для регистрации в качестве пользователя в windows Insider.

В настоящее время в "Каналы" переходит внутренняя пользователи Windows. Круг **"Быстрый"** станет каналом **Dev,** круг **Slow** станет **** каналом **Beta,** а круг предварительного просмотра выпуска станет каналом Предварительного просмотра **выпуска.** Вот как выглядит это сопоставление:

![Описание каналов для insider Windows](images/WindowsInsiderChannels.png)

Дополнительные сведения см. в [канале "Знакомство с Windows Insider Channels" в](https://blogs.windows.com/windowsexperience/2020/06/15/introducing-windows-insider-channels) блогах Windows.

Затем выберите **"Активная разработка Для Windows",** выберите, хотите ли вы получать сборки канала **Dev** или **Beta Channel,** и просмотрите условия программы.

Выберите **"> перезапустить сейчас",** чтобы завершить работу. После перезагрузки устройства перейдите в **** > "& Безопасность >", чтобы получить последнюю сборку.

## Маршруты загрузки и вспышки FFU
Чтобы протестировать ffu со подписью тестового устройства, сначала необходимо разблокировать устройство, прежде чем мигать ffu с подписью тестового устройства.
1. На компьютере:

    1. Скачайте ffu на компьютер с [https://aka.ms/hololenspreviewdownload](https://aka.ms/hololenspreviewdownload) .
    
    1. Установите ARC (advanced Recovery Companion) из Microsoft Store: [https://www.microsoft.com/store/productId/9P74Z35SFRS8](https://www.microsoft.com/store/productId/9P74Z35SFRS8)
    
1. On HoloLens - Flight Unlock: Open **Settings**  >  **Update & Security**Windows  >  **Insider Program** then sign up, reboot device.

1. Flash FFU — теперь вы можете мигать FFU с подписью в полете с помощью ARC.

## Прекратить получение сборки для insider

Если вы больше не хотите получать сборки Windows Holographic для предварительной точки, вы можете отказаться [](hololens-recovery.md) от использования производственной сборки HoloLens или восстановить устройство с помощью устройства Advanced Recovery Companion, чтобы восстановить устройство до версии Windows Holographic, которая не является предварительной.

> [!CAUTION]
> Существует известная проблема, из-за которой пользователи, которые отостановят регистрацию в сборках Insider Preview после ручной переустановки новой сборки предварительного просмотра, будут испытывать синий экран. После этого им необходимо вручную восстановить устройство. Дополнительные сведения о том, как это повлияет на вас, можно просмотреть по этой [известной проблеме.](https://docs.microsoft.com/hololens/hololens-known-issues?source=docs#blue-screen-is-shown-after-unenrolling-from-insider-preview-builds-on-a-device-reflashed-with-a-insider-build)

Чтобы убедиться, что holoLens работает под управлением производственной сборки:

1. Перейдите **в > > "О**системе" и найдите номер сборки.

1. [См. заметки о выпуске производственных номеров сборки.](hololens-release-notes.md)

Чтобы отказаться от сборки для insider:

1. On a HoloLens running a production build, go to **Settings > Update & Security > Windows Insider Program**, and select **Stop Insider builds**.

1. Следуйте инструкциям, чтобы отказаться от устройства.
