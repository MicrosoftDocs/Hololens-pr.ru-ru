# Использование голосовых команд для работы с HoloLens

Вы можете использовать голосовые команды для выполнения практически любых действий на HoloLens, таких как быстрая фотосъемка или открытие приложения. Многие голосовые команды встроены в HoloLens, а другие доступны с помощью Кортаны.

В этой статье рассказывается о том, как управлять HoloLens и голографическим миром с помощью голосовых команд и Кортаны.

> [!NOTE]
> Голосовые функции поддерживаются только на [некоторых языках](hololens2-language-support.md). Язык голосовых функций зависит от языка интерфейса Windows, а не языка клавиатуры.  
>  
> Чтобы узнать язык интерфейса Windows, выберите **Параметры** > **Время и язык** > **Язык**.

## Встроенные голосовые команды

Работайте с HoloLens быстрее с помощью этих основных команд. Чтобы воспользоваться ими, при первом запуске устройства включите "Речь" или выберите **Настройки** > **Конфиденциальность** > **Речь**. Вы всегда можете узнать, включена ли речь, проверив состояние в верхней части меню "Пуск". Для достижения наилучших результатов распознавания речи в HoloLens2 используются облачные службы Майкрософт. Однако вы можете отключить эту функцию в разделе "Настройки". Чтобы сделать это, в разделе "Настройки" отключите **Распознавание речи в сети**. После изменения этого параметра в HoloLens 2 голосовые данные при распознавании команд и диктовке будут обрабатываться только локально, а Кортана будет недоступна.

### Общие голосовые команды

Чтобы быстрее приступить к работе, используйте эти команды во всей среде Windows Mixed Reality. В некоторых командах используется курсор взгляда, который можно открыть, сказав "select" (выбрать).

> [!NOTE]
> Лучи рук не поддерживаются в HoloLens (1-е поколение).

| Команда | Действие |
| - | - |
| "Select" | Произнесите "select" (выбрать) для отображения курсора взгляда. Затем поверните голову так, чтобы курсор указывал на элемент, который требуется выбрать, и снова произнесите "select". |
|Open the Start menu | "Go to Start" ("Перейди в меню "Пуск""). |
|Close the Start menu | "Close" (Закрыть) |
|Leave an immersive app | Скажите "Go to Start" (Перейди в меню "Пуск"), чтобы открыть меню быстрых действий, а затем скажите "Mixed reality home" (Домашняя смешанная реальность). |
|Hide and show hand ray | "Hide hand ray" (Скрыть лучи рук)/"Show hand ray" (Показать лучи рук) |
|See available speech commands | "What can I say?" (Что можно говорить?) |

Начиная с версии19041.x в HoloLens2 можно использовать следующие команды.

| Команда | Действие |
| - | - |
| "Restart device" | Отображение диалогового окна для подтверждения перезапуска устройства. Скажите "yes" (да), чтобы перезапустить. |
| "Shutdown device" | Отображение диалогового окна для подтверждения выключения устройства. Скажите "yes" (да), чтобы подтвердить. |
| "Brightness up/down" | Увеличение или уменьшение яркости дисплея на 10%. |
| "Volume up/down" | Увеличение или уменьшение громкости на 10%. |
| "What's my IP address" | Отображение диалогового окна, в котором указан текущий IP-адрес вашего устройства в локальной сети. |
| "Take a picture" | Захват изображения, которое вы наблюдаете в смешанной реальности в данный момент. |
| "Take a video" | Начало записи видео в смешанной реальности. | 
| "Stop recording" | Завершение начатой записи видео в смешанной реальности. |

### Команды с голограммами

Чтобы использовать эти команды, посмотрите на трехмерный объект, голограмму или окно приложения.

| Команда | Задача |
| - | - |
| "Bigger" | Увеличить |
| "Smaller" | Уменьшить |
| "Face me" | Повернуть лицом к вам |
| "Move this" | Переместить (по взгляду) |
| "Close" | Закрыть |
| "Follow me" / "Stop following" | Объект следует за вами, когда вы перемещаетесь |

### Увидеть и сказать

Многие кнопки и другие элементы в HoloLens также реагируют на ваш голос, например **Follow me** (Следовать за мной) и **Close** (Закрыть) на панели приложения или кнопка **Назад** в Edge. Чтобы узнать, включена ли кнопка с голосовой командой, наведите **курсор на взгляд**, **коснитесь курсора** или на одну **руку** на мгновение. Если кнопка включена голосом, вы увидите голосовой совет.

### Режим диктовки

Устали вводить текст? Перейдите в режим диктовки в любое время, чтобы активировать голографическую клавиатуру. Чтобы начать работу, нажмите кнопку микрофона или скажите "Start dictating" (Начать диктовку). Чтобы прекратить диктовку, нажмите эту кнопку снова или скажите "Stop dictating" (Прекратить диктовку). Чтобы удалить только что продиктованное слово, скажите "Delete that" (Удалить это). 

> [!NOTE]
> Для использования режима диктовки требуется подключение к Интернету.

Функция диктовки HoloLens использует явные знаки препинания, то есть вам необходимо говорить знак препинания, который вы хотите использовать. Например, вы можете сказать "Привет, **запятая** , как дела **вопросительный знак**".

Ниже перечислены ключевые слова знаков пунктуации, которые можно использовать.

- Точка, запятая, вопрос, восклицательный знак
- Новая строка/новый абзац
- Точка с запятой, двоеточие
- Открыть кавычки, закрыть кавычки
- Хэштег, смайлик, хмурый смайлик, подмигивающий смайлик
- Доллар, процент

Иногда бывает полезно говорить такие элементы, как адреса электронной почты, по буквам. Например, чтобы продиктовать example@outlook.com, вы можете сказать "E X A M P L E собака outlook точка com".

## Делайте больше с помощью Кортаны

С помощью Кортаны в HoloLens можно делать много всего интересного, но в зависимости от используемой версии Windows Holographic доступные возможности могут отличаться. Подробные сведения о новых возможностях последней версии Кортаны приведены [здесь](https://blogs.windows.com/windowsexperience/2020/02/28/cortana-in-the-upcoming-windows-10-release-focused-on-your-productivity-with-enhanced-security-and-privacy/). 

![Привет, Кортана!](images/cortana-on-hololens.png)

Вот несколько фраз, которые вы можете сказать (сначала скажите "Привет, Кортана!").

**Hey, Cortana**...

- What can I say?
- Launch <*имя приложения*>.
- What time is it?
- Show me the latest NBA scores.
- Tell me a joke.

В *версии 18362.x и более ранних* можно также использовать следующие команды.

**Hey, Cortana**...

- Increase the volume.
- Decrease the brightness.
- Shut down.
- Restart.
- Go to sleep.
- Mute.
- Move <*имя приложения*> here (взгляните на место, куда вы хотите переместить приложение).
- Go to Start.
- Take a picture.
- Start recording. (Начинает запись видео.)
- Stop recording. (Остановка записи видео.)
- How much battery do I have left?

Некоторые функции Кортаны, которые вы используете в Windows на компьютере или телефоне (например, напоминания и уведомления), не поддерживаются в Microsoft HoloLens, а возможности Кортаны зависят от региона.

### Отключение Кортаны

Кортана включается при первом использовании HoloLens при активации голосовых функций. Кортану можно отключить в разделе параметров Кортаны. В списке **Все приложения** выберите **Кортана** > **Параметры**. Затем отключите параметры Кортаны, чтобы перестать получать рекомендации, идеи, напоминания, оповещения и многое другое.

Если Кортана не отвечает на фразу "Привет, Кортана!", убедитесь, что голосовые функции включены в меню "Пуск", и перейдите в параметры Кортаны, чтобы убедиться, что она включена.
